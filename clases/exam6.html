EXAMEN 6

Servicios IP (DHCP, NAT/PAT, ACL, DNS, NTP, Syslog)

⏱️ Sin límite de tiempo 📝 30 preguntas ✅ Respuestas múltiples permitidas
Pregunta 1
¿Qué puerto UDP utiliza DHCP para la comunicación cliente→servidor?
✓ Respuesta Correcta: B
El cliente DHCP usa puerto UDP 68 y el servidor usa 67. Cliente envía DHCPDISCOVER desde :68 → :67. Servidor responde DHCPOFFER desde :67 → :68.
Pregunta 2
¿Cuál es el orden correcto del proceso DHCP?
✓ Respuesta Correcta: A
Proceso DORA: Discover → Offer → Request → Acknowledge. Cliente envía broadcast Discover, servidor(es) responden Offer, cliente selecciona uno con Request broadcast, servidor confirma con Acknowledge unicast.
Pregunta 3
¿Qué tipo de NAT permite que múltiples hosts internos compartan una sola IP pública usando puertos?
✓ Respuesta Correcta: C
PAT (Port Address Translation) o NAT overload usa una IP pública con múltiples puertos para mapear muchos hosts internos. Static NAT = 1:1, Dynamic NAT = N:M (pool), PAT = N:1 con puertos.
Pregunta 4
¿Qué comando configura un router Cisco como DHCP Relay Agent?
✓ Respuesta Correcta: B
ip helper-address [IP_servidor_DHCP] convierte broadcast DHCP en unicast dirigido al servidor remoto. Se configura en la interfaz donde llegan los DHCPDISCOVER de clientes.
Pregunta 5
En ACL, ¿qué significa una wildcard mask de 0.0.0.255?
✓ Respuesta Correcta: C
Wildcard 0.0.0.255: 0 = debe coincidir, 1 = ignorar. Los primeros 3 octetos deben coincidir exactamente, el 4to puede ser cualquier valor (0-255). Ejemplo: 10.1.1.0 0.0.0.255 = toda la red 10.1.1.0/24.
Pregunta 6
¿Cuál es la diferencia principal entre ACL estándar y ACL extendida?
✓ Respuesta Correcta: A
ACL Estándar (1-99, 1300-1999): filtra solo por IP origen. ACL Extendida (100-199, 2000-2699): filtra por origen, destino, protocolo (TCP/UDP/ICMP), puertos, flags TCP, etc. La B es correcta pero A es más completa.
Pregunta 7
¿Qué función realiza DNS?
✓ Respuesta Correcta: B
DNS (Domain Name System) resuelve nombres → IPs. Usa UDP/53 para queries normales, TCP/53 para transferencias de zona. Jerarquía: Root (.) → TLD (.com) → Authoritative (google.com).
Pregunta 8
¿Qué puerto y protocolo usa NTP?
✓ Respuesta Correcta: B
NTP (Network Time Protocol) usa UDP/123. Sincroniza relojes con precisión de milisegundos. Niveles (stratum): 0=reloj atómico, 1=servidor primario, 2-15=secundarios. SNMP=161, Syslog=514.
Pregunta 9
¿Dónde se recomienda aplicar una ACL estándar?
✓ Respuesta Correcta: B
Estándar: cerca del destino (filtra solo origen, aplicarla cerca del origen bloquearía todo tráfico de esa fuente). Extendida: cerca del origen (granular, evita tráfico innecesario en la red).
Pregunta 10
¿Qué comando aplica una ACL a una interfaz para filtrar tráfico entrante?
✓ Respuesta Correcta: B
ip access-group [ACL] {in|out} aplica ACL en interfaz. in=filtra paquetes entrantes (antes de routing), out=filtra salientes (después de routing). access-class es para VTY lines.
Pregunta 11
En NAT, ¿qué dirección se considera "inside local"?
✓ Respuesta Correcta: A
Terminología NAT: Inside Local=IP privada original, Inside Global=IP pública traducida, Outside Local=IP externa vista desde inside, Outside Global=IP real externa.
Pregunta 12
¿Qué protocolo permite enviar mensajes de log desde dispositivos de red a un servidor centralizado?
✓ Respuesta Correcta: B
Syslog (UDP/514) envía mensajes de eventos/logs a servidor centralizado. Niveles: 0=Emergency, 1=Alert, 2=Critical, 3=Error, 4=Warning, 5=Notice, 6=Informational, 7=Debug. SNMP=monitoreo/gestión.
Pregunta 13
¿Qué ACL entry permite todo el tráfico ICMP?
✓ Respuesta Correcta: B
Sintaxis ACL extendida: access-list [100-199] {permit|deny} [protocolo] [origen] [destino]. Para ICMP específico: agregar tipo (ej. echo, echo-reply, unreachable).
Pregunta 14
¿Cuál de los siguientes NO es un mensaje DHCP válido?
✓ Respuesta Correcta: D
Mensajes DHCP válidos: DISCOVER, OFFER, REQUEST, ACK (DORA), DECLINE, RELEASE, INFORM, NAK. DHCPCONFIRM no existe en DHCPv4 (existe en DHCPv6).
Pregunta 15
¿Qué comando configura NAT estático (1:1)?
✓ Respuesta Correcta: A
NAT estático: ip nat inside source static [IP_local] [IP_global]. B=pool para dynamic NAT. C=PAT con overload. Recordar: marcar interfaces con ip nat {inside|outside}.
Pregunta 16
En ACL, ¿qué efecto tiene la regla implícita al final?
✓ Respuesta Correcta: B
Todas las ACL terminan con implicit deny any invisible. Si un paquete no coincide con ninguna entrada, es denegado. Best practice: agregar permit ip any any log explícito al final para debugging.
Pregunta 17
¿Qué stratum tiene un dispositivo NTP directamente conectado a un reloj atómico GPS?
✓ Respuesta Correcta: B
Stratum 0=reloj de referencia (GPS, atómico) no conectado a red. Stratum 1=servidor NTP directamente conectado a stratum 0. Stratum 2=sincronizado desde stratum 1, etc. Máximo=15 (16=unsynchronized).
Pregunta 18
¿Qué parámetro DHCP excluye un rango de IPs del pool de asignación?
✓ Respuesta Correcta: A
ip dhcp excluded-address [inicio] [fin] (modo config global) excluye IPs de asignación dinámica. Típicamente usado para gateway, servidores DNS, impresoras. Se configura ANTES de definir el pool.
Pregunta 19
¿Qué puerto TCP usa DNS para transferencias de zona?
✓ Respuesta Correcta: B
DNS usa UDP/53 para queries normales (rápido, sin conexión). TCP/53 se usa para: transferencias de zona (AXFR/IXFR), respuestas >512 bytes, y comunicación entre servidores autoritativos.
Pregunta 20
¿Qué comando verifica las traducciones NAT activas?
✓ Respuesta Correcta: B
show ip nat translations muestra tabla de traducciones activas: Inside local/global, Outside local/global. show ip nat statistics muestra contadores y configuración. clear ip nat translation * limpia tabla.
Pregunta 21
¿Qué nivel de severidad Syslog representa mensajes de "Error"?
✓ Respuesta Correcta: B
Syslog levels: 0=Emergency, 1=Alert, 2=Critical, 3=Error, 4=Warning, 5=Notice, 6=Informational, 7=Debug. logging trap [level] configura qué enviar al servidor.
Pregunta 22
¿Qué ACL extendida permite HTTP (puerto 80) desde cualquier origen a 10.0.0.5?
✓ Respuesta Correcta: C
Ambas sintaxis son válidas: eq 80 o eq www (nombre de servicio). host 10.0.0.5 equivale a 10.0.0.5 0.0.0.0. Otros: eq 443 / eq https, eq 22 / eq ssh.
Pregunta 23
¿Qué opción DHCP configura el lease time (tiempo de arrendamiento)?
✓ Respuesta Correcta: C
Dentro del pool DHCP: lease [días] [horas] [minutos] o lease infinite para lease permanente. Default=1 día. Cliente intenta renovar al 50% del lease (T1), luego al 87.5% (T2).
Pregunta 24
¿Qué hace el comando debug ip nat?
✓ Respuesta Correcta: A
debug ip nat muestra en consola cada traducción NAT en tiempo real (origen→traducido, destino→traducido). ⚠️ Alto impacto en CPU, usar con cuidado. undebug all para desactivar.
Pregunta 25
¿Cuál es la función de "remark" en ACL?
✓ Respuesta Correcta: B
access-list [num] remark [texto] o en ACL nombrada: remark [texto]. Agrega comentarios para documentar propósito de cada regla. Best practice: remark antes de cada entrada lógica.
Pregunta 26
¿Qué sucede cuando un cliente DHCP no recibe DHCPOFFER?
✓ Respuesta Correcta: B
Sin OFFER, el cliente reintenta DISCOVER con backoff exponencial (típicamente 4 intentos). Si falla, asigna IP APIPA (169.254.0.0/16) y sigue intentando cada 5 minutos.
Pregunta 27
En PAT, ¿qué diferencia hace que múltiples hosts internos puedan compartir una IP pública?
✓ Respuesta Correcta: A
PAT traduce: [IP_privada:puerto_origen] → [IP_pública:puerto_traducido]. La tupla única (IP_pública + puerto) identifica cada sesión. Soporta ~65,000 conexiones simultáneas por IP pública (límite de puertos).
Pregunta 28
¿Qué comando configura un router Cisco como cliente NTP?
✓ Respuesta Correcta: A
ntp server [IP_servidor] configura router como cliente NTP. ntp master [stratum] lo hace servidor autoritativo. Verificar: show ntp status, show ntp associations.
Pregunta 29
¿Qué hace la keyword "log" al final de una ACL entry?
✓ Respuesta Correcta: C
log genera mensajes cuando hay match, mostrando: ACL#, acción, protocolo, origen, destino. log-input incluye además interfaz de entrada. ⚠️ Alto volumen puede impactar CPU. Útil para debugging/seguridad.
Pregunta 30
¿Qué opción DHCP proporciona la dirección IP del gateway por defecto?
✓ Respuesta Correcta: B
DHCP Options: 3=default-router (gateway), 1=subnet-mask, 6=dns-server, 15=domain-name, 42=ntp-server, 66=tftp-server, 150=tftp-server (Cisco). En config pool: default-router 192.168.1.1.
