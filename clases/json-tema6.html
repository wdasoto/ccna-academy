Proyecto — Inventario Automatizado en JSON

Genera automáticamente un inventario de red completo en formato JSON: dispositivos, interfaces, IPs, versiones IOS y estado. Conexión real vía Netmiko, serialización profesional con json.dumps y persistencia en archivos.

🗂️ Proyecto Final 🎯 CCNA Domain 6.2 ⏱ ~120 min desarrollo 💾 Archivo .json real 🔌 Netmiko Live

El Golden JSON: Tu Fuente de Verdad

En infraestructuras modernas, el inventario de red no vive en hojas de Excel estáticas ni en documentos Word desactualizados. Vive en un archivo JSON que se regenera automáticamente cada hora, consumido por sistemas de monitoreo, automatización y orquestación.

Este proyecto es el pilar de NetDevOps: extraes datos reales de dispositivos Cisco mediante SSH (Netmiko), los estructuras en diccionarios Python, los serializas a JSON legible y lo persistes en disco. El resultado es un single source of truth que alimenta cualquier sistema downstream.

📊

Estructura Anidada

Diseña un schema JSON que incluya dispositivos, interfaces, IPs y metadatos de versiones IOS.

⚡

Extracción Real

Conecta a routers y switches vía Netmiko para obtener show commands y parsear outputs.

💾

Persistencia

Usa json.dumps() con indentación y open() para guardar en disco.

🔄

Automatización

Genera el inventario periódicamente vía cron/scheduler para mantener datos frescos.

💡 Arquitectura del Proyecto

Entrada: Lista de dispositivos (IP, credenciales)
Proceso: Conexión SSH → Comandos show → Parseo → Diccionarios
Salida: Archivo inventario_red.json con timestamp y estructura jerárquica

Antes de escribir código de red, diseñamos el esquema de datos. En Python, esto se traduce en diccionarios anidados y listas. Cada dispositivo es un dict; las interfaces son una lista de dicts dentro del dispositivo.

Extracción de Datos con Netmiko

Conectamos a los dispositivos y ejecutamos comandos específicos para poblar nuestro schema. Usamos textFSM (implícito en Netmiko) o parseo manual para estructurar los outputs.

cisco_collector.py — Obtención de datos estructurados
from netmiko import ConnectHandler
import re

def obtener_info_basica(conn):
    """Extrae hostname, versión y modelo de show version"""
    output = conn.send_command("show version")
    
    info = {}
    # Parseo con regex para IOS-XE / IOS
    hostname_match = re.search(r'hostname\s+(\S+)', output, re.I)
    if hostname_match:
        info["hostname"] = hostname_match.group(1)
    
    version_match = re.search(r'Version\s+(\d+\.\d+)', output)
    if version_match:
        info["version_ios"] = version_match.group(1)
    
    modelo_match = re.search(r'cisco\s+(\S+).*processor', output, re.I)
    if modelo_match:
        info["modelo"] = modelo_match.group(1)
    
    return info

def obtener_interfaces_json(conn):
    """Extrae interfaces de show ip interface brief"""
    output = conn.send_command("show ip interface brief", use_textfsm=True)
    # Netmiko con textFSM devuelve lista de diccionarios directamente
    interfaces = []
    
    for intf in output:
        interfaces.append({
            "nombre": intf["intf"],
            "ip": intf["ipaddr"] if intf["ipaddr"] else "unassigned",
            "estado_linea": intf["status"],
            "protocolo": intf["proto"]
        })
    return interfaces

# Conexión y recolección completa
device_params = {
    "device_type": "cisco_ios",
    "host": "192.168.1.1",
    "username": "admin",
    "password": "Cisco123",
    "secret": "EnableSecret"
}

try:
    with ConnectHandler(**device_params) as conn:
        conn.enable()  # Entrar a enable mode
        datos_basicos = obtener_info_basica(conn)
        interfaces = obtener_interfaces_json(conn)
        
        print(f"✅ {datos_basicos['hostname']} recolectado")
except Exception as e:
    print(f"❌ Error en {device_params['host']}: {e}")
⚠️ Manejo de Errores de Conexión

Siempre envuelve las conexiones en try/except. Un dispositivo offline no debe detener el inventario completo. Guarda el error en error_msg y continúa con el siguiente dispositivo.

De Python a JSON: json.dumps()

Python tiene un módulo JSON robusto. La función json.dumps() convierte diccionarios a strings JSON, mientras que json.dump() escribe directamente a archivo.

serializacion.py — Control total del formato
import json

# Datos de ejemplo
datos = {
    "dispositivos": [
        {"hostname": "R1", "ip": "10.0.0.1", "activo": True},
        {"hostname": "R2", "ip": "10.0.0.2", "activo": False}
    ]
}

# Opción 1: Convertir a string (para imprimir o enviar por API)
json_string = json.dumps(datos)
print(json_string)  # Una línea compacta

# Opción 2: Pretty print con indentación
json_pretty = json.dumps(datos, indent=2)
print(json_pretty)

# Opción 3: Ordenar claves alfabéticamente (consistente)
json_ordenado = json.dumps(datos, indent=2, sort_keys=True)

# Opción 4: Fechas datetime (no serializables por defecto)
from datetime import datetime

def json_serial(obj):
    """JSON serializer para objetos no serializables por defecto"""
    if isinstance(obj, datetime):
        return obj.isoformat()
    raise TypeError("Type not serializable")

# Uso con función personalizada de serialización
datos_con_fecha = {"timestamp": datetime.now(), "data": "test"}
json_con_fecha = json.dumps(datos_con_fecha, default=json_serial, indent=2)
💡 Parámetros Críticos de json.dumps()
  • indent=2: Formato legible con sangría de 2 espacios
  • sort_keys=True: Claves ordenadas alfabéticamente (útil para diffs)
  • ensure_ascii=False: Permite caracteres UTF-8 (tildes, ñ)
  • default=func: Handler para tipos no serializables (datetime, set, etc.)

Persistencia: Escribiendo el Archivo JSON

Una vez serializado, guardamos el JSON en disco con manejo seguro de archivos (context manager with) y codificación UTF-8.

file_manager.py — Gestión profesional de archivos
import json
import os
from datetime import datetime

def guardar_inventario(datos, directorio="./output", prefijo="inventario"):
    """
    Guarda el inventario en un archivo JSON con timestamp.
    Retorna la ruta del archivo creado.
    """
    # Crear directorio si no existe
    if not os.path.exists(directorio):
        os.makedirs(directorio)
        print(f"📁 Directorio creado: {directorio}")
    
    # Generar nombre de archivo con timestamp
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    filename = f"{prefijo}_{timestamp}.json"
    filepath = os.path.join(directorio, filename)
    
    # Escribir archivo con formato profesional
    with open(filepath, 'w', encoding='utf-8') as f:
        json.dump(
            datos, 
            f, 
            indent=2, 
            sort_keys=True, 
            ensure_ascii=False
        )
    
    print(f"💾 Inventario guardado: {filepath}")
    return filepath

# Función complementaria: Leer inventario
def cargar_inventario(filepath):
    """Carga un inventario desde archivo JSON"""
    with open(filepath, 'r', encoding='utf-8') as f:
        return json.load(f)  # json.load lee desde archivo
✅ Validación de Archivos

Siempre usa encoding='utf-8' para soportar caracteres internacionales. El modo 'w' sobrescribe; usa 'a' (append) solo si quieres agregar a un archivo existente sin borrar.

🚀 Proyecto Final — Generador de Inventario JSON

Este es el script completo de producción. Conecta a múltiples dispositivos, maneja errores individualmente, estructura todo en JSON jerárquico y guarda el archivo timestamped.

generar_inventario.py — Solución completa (150 líneas)
#!/usr/bin/env python3
"""
generar_inventario.py
─────────────────────────────────────────────────
Genera inventario JSON automatizado de red Cisco.
Requiere: pip install netmiko
─────────────────────────────────────────────────
"""

import json
import os
from datetime import datetime
from netmiko import ConnectHandler
from netmiko.ssh_exception import NetmikoTimeoutException, AuthenticationException

# ══════════════════════════════════════════════════════
# CONFIGURACIÓN: Define tus dispositivos aquí
# ══════════════════════════════════════════════════════
DISPOSITIVOS_CONFIG = [
    {
        "device_type": "cisco_ios",
        "host": "192.168.1.1",
        "username": "admin",
        "password": "Cisco123",
        "secret": "EnablePass",
        "hostname_esperado": "R1-CORE"
    },
    {
        "device_type": "cisco_ios",
        "host": "192.168.1.2",
        "username": "admin",
        "password": "Cisco123",
        "hostname_esperado": "SW1-ACC"
    },
    {
        "device_type": "cisco_ios",
        "host": "192.168.1.99",  # Simulación de dispositivo offline
        "username": "admin",
        "password": "WrongPass",
        "hostname_esperado": "R2-BACKUP"
    }
]

# ══════════════════════════════════════════════════════
# FUNCIONES DE COLECCIÓN
# ══════════════════════════════════════════════════════

def colectar_dispositivo(dev_config):
    """Conecta a un dispositivo y extrae toda la información"""
    
    resultado = {
        "ip_mgmt": dev_config["host"],
        "hostname": dev_config.get("hostname_esperado", "unknown"),
        "estado_conexion": "pending",
        "error_msg": None,
        "version_ios": None,
        "modelo": None,
        "uptime": None,
        "interfaces": [],
        "rutas_totales": 0
    }
    
    try:
        print(f"🔌 Conectando a {dev_config['host']}...")
        
        with ConnectHandler(**dev_config) as conn:
            conn.enable()
            
            # 1. Hostname real (verificación)
            prompt = conn.find_prompt().strip('#').strip('>')
            resultado["hostname"] = prompt
            
            # 2. Versión y modelo (parseo simple)
            show_ver = conn.send_command("show version")
            if "Version" in show_ver:
                import re
                ver_match = re.search(r'Version\s+(\d+\.[\d\w\.]+)', show_ver)
                if ver_match:
                    resultado["version_ios"] = ver_match.group(1)
                
                model_match = re.search(r'cisco\s+(\S+).*with.*bytes', show_ver, re.I)
                if model_match:
                    resultado["modelo"] = model_match.group(1)
            
            # 3. Interfaces (usando TextFSM si disponible)
            try:
                show_ip = conn.send_command("show ip interface brief", use_textfsm=True)
                if isinstance(show_ip, list):
                    for intf in show_ip:
                        resultado["interfaces"].append({
                            "nombre": intf.get("intf", "unknown"),
                            "ip": intf.get("ipaddr") or "unassigned",
                            "estado": intf.get("status", "unknown"),
                            "protocolo": intf.get("proto", "unknown")
                        })
            except Exception as e:
                resultado["interfaces"] = [{"error": "No se pudieron obtener interfaces"}]
            
            # 4. Conteo de rutas
            show_route = conn.send_command("show ip route | count")
            if "Route" in show_route:
                import re
                count_match = re.search(r'(\d+)', show_route)
                if count_match:
                    resultado["rutas_totales"] = int(count_match.group(1))
            
            resultado["estado_conexion"] = "success"
            print(f"   ✅ {resultado['hostname']} recolectado")
            
    except AuthenticationException:
        resultado["estado_conexion"] = "auth_failed"
        resultado["error_msg"] = "Autenticación fallida"
        print(f"   ❌ Auth failed en {dev_config['host']}")
    except NetmikoTimeoutException:
        resultado["estado_conexion"] = "timeout"
        resultado["error_msg"] = "Dispositivo no responde"
        print(f"   ❌ Timeout en {dev_config['host']}")
    except Exception as e:
        resultado["estado_conexion"] = "error"
        resultado["error_msg"] = str(e)
        print(f"   ❌ Error en {dev_config['host']}: {e}")
    
    return resultado

# ══════════════════════════════════════════════════════
# FUNCIÓN PRINCIPAL
# ══════════════════════════════════════════════════════

def main():
    print("\n" + "="*60)
    print("  CCNA ACADEMY — GENERADOR DE INVENTARIO JSON")
    print("="*60 + "\n")
    
    # Inicializar estructura maestra
    inventario = {
        "metadata": {
            "timestamp": datetime.now().isoformat(),
            "generado_por": "ccna_inventory_generator/2.0",
            "total_intentos": len(DISPOSITIVOS_CONFIG)
        },
        "resumen": {
            "exitosos": 0,
            "fallidos": 0,
            "total_interfaces": 0
        },
        "dispositivos": []
    }
    
    # Procesar cada dispositivo
    for dev_conf in DISPOSITIVOS_CONFIG:
        datos = colectar_dispositivo(dev_conf)
        inventario["dispositivos"].append(datos)
        
        if datos["estado_conexion"] == "success":
            inventario["resumen"]["exitosos"] += 1
            inventario["resumen"]["total_interfaces"] += len(datos["interfaces"])
        else:
            inventario["resumen"]["fallidos"] += 1
    
    # Guardar a archivo
    output_dir = "./inventarios_json"
    if not os.path.exists(output_dir):
        os.makedirs(output_dir)
    
    filename = f"inventario_red_{datetime.now().strftime('%Y%m%d_%H%M%S')}.json"
    filepath = os.path.join(output_dir, filename)
    
    with open(filepath, 'w', encoding='utf-8') as f:
        json.dump(inventario, f, indent=2, sort_keys=True, ensure_ascii=False)
    
    # Reporte final
    print("\n" + "="*60)
    print("  RESUMEN DE EJECUCIÓN")
    print("="*60)
    print(f"  ✅ Exitosos:   {inventario['resumen']['exitosos']}")
    print(f"  ❌ Fallidos:   {inventario['resumen']['fallidos']}")
    print(f"  📊 Interfaces: {inventario['resumen']['total_interfaces']}")
    print(f"\n💾 Archivo guardado: {filepath}")
    print(="*60 + "\n")

if __name__ == "__main__":
    main()
