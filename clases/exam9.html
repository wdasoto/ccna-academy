EXAMEN 9

Wireless & Automatización de Redes

⏱️ Sin límite📝 50 preguntas
Pregunta 1
¿Qué estándar 802.11 opera en la banda de 5 GHz y alcanza hasta 1.3 Gbps?
✓ Respuesta: D
802.11ac (Wi-Fi 5): 5 GHz, hasta 1.3 Gbps (MU-MIMO, 80/160 MHz). 802.11n=600 Mbps (2.4/5 GHz), 802.11ax (Wi-Fi 6)=9.6 Gbps.
Pregunta 2
¿Qué protocolo de gestión de configuración usa modelo declarativo y playbooks YAML?
✓ Respuesta: C
Ansible: agentless, YAML playbooks, push model, SSH. Puppet/Chef=agent-based, pull model. Ansible es el más usado para redes.
Pregunta 3
¿Qué modo de deployment wireless centraliza todas las funciones en el WLC (Wireless LAN Controller)?
✓ Respuesta: B
Lightweight AP con WLC: APs "tontos" (CAPWAP tunnel al WLC), gestión centralizada. Autonomous=standalone, FlexConnect=local switching en sucursales.
Pregunta 4
¿Qué formato de datos se usa comúnmente en REST APIs para configuración de red?
✓ Respuesta: D
REST APIs usan JSON (más común, liviano) o XML. NETCONF usa XML. YAML se usa en Ansible playbooks.
Pregunta 5
¿Qué protocolo permite configuración programática de dispositivos de red sobre SSH?
✓ Respuesta: B
NETCONF (RFC 6241): protocolo sobre SSH/TLS, usa XML, modelo YANG, transaccional. Puerto 830. RESTCONF es alternativa sobre HTTPS.
Pregunta 6
¿Qué canal wireless de 2.4 GHz NO se superpone con canal 1 y 6?
✓ Respuesta: B
2.4 GHz: solo 3 canales no-solapados: 1, 6, 11 (22 MHz de ancho c/u). 5 GHz tiene 24 canales no-solapados (20/40/80/160 MHz).
Pregunta 7
¿Qué modelo de red SDN separa plano de control del plano de datos?
✓ Respuesta: B
SDN: controlador centralizado (plano control), switches/routers solo forwarding (plano datos). OpenFlow, Cisco ACI, DNA Center son ejemplos.
Pregunta 8
¿Qué protocolo usa CAPWAP para túneles entre AP y WLC?
✓ Respuesta: B
CAPWAP (Control And Provisioning of Wireless Access Points): UDP/5246 (control) y UDP/5247 (data). Reemplazó LWAPP.
Pregunta 9
¿Qué herramienta de Cisco usa intención (intent-based networking) para automatizar red?
✓ Respuesta: A
DNA Center: IBN (Intent-Based Networking), automatización campus/branch, telemetría, segmentación, políticas. APIC=data center (ACI).
Pregunta 10
¿Qué método HTTP REST se usa para crear un nuevo recurso?
✓ Respuesta: B
CRUD REST: POST=Create, GET=Read, PUT/PATCH=Update, DELETE=Delete. POST crea recurso nuevo, PUT actualiza existente.
Pregunta 11
¿Qué tipo de BSS (Basic Service Set) usa múltiples APs con mismo SSID?
✓ Respuesta: B
ESS = múltiples APs + mismo SSID = roaming transparente. BSS = 1 AP. IBSS = peer-to-peer sin AP.
Pregunta 12
¿Qué lenguaje de modelado de datos se usa con NETCONF/RESTCONF?
✓ Respuesta: C
YANG (Yet Another Next Generation) define modelos de datos. NETCONF transporta XML según schema YANG. RESTCONF puede usar JSON/XML.
Pregunta 13
¿Qué arquitectura de red inalámbrica utiliza Lightweight Access Points (LAPs) gestionados por una WLC central?
✓ Respuesta: B
La arquitectura Split-MAC divide las funciones de la capa MAC entre el LAP y el controlador (WLC), permitiendo la gestión centralizada.
Pregunta 14
En una infraestructura Cisco, ¿cuál es el protocolo encargado de encapsular el tráfico de datos y control entre un LAP y una WLC?
✓ Respuesta: B
CAPWAP utiliza los puertos UDP 5246 (control) y 5247 (datos) para esta comunicación.
Pregunta 15
¿Qué verbo HTTP se utiliza en una API REST para crear un nuevo recurso en el servidor?
✓ Respuesta: C
El método POST se utiliza para enviar datos al servidor con el fin de crear un nuevo recurso.
Pregunta 16
¿Cuál es una característica principal de las redes WLAN que operan en la banda de 5 GHz comparada con la de 2.4 GHz?
✓ Respuesta: B
La banda de 5 GHz ofrece muchos más canales y menos congestión que la banda de 2.4 GHz, que solo tiene 3 canales sin superposición (1, 6 y 11).
Pregunta 17
En la arquitectura SDN, ¿cómo se denomina a la interfaz que permite la comunicación entre el controlador y las aplicaciones de red?
✓ Respuesta: B
Las Northbound APIs conectan el controlador con aplicaciones de nivel superior para la gestión y monitoreo.
Pregunta 18
¿Qué tipo de antena se recomienda para proporcionar cobertura en un pasillo largo y estrecho de un almacén?
✓ Respuesta: B
Las antenas Direccionales (como la Patch o Yagi) enfocan la energía en una dirección específica, ideal para pasillos o áreas largas.
Pregunta 19
¿Qué herramienta de automatización utiliza "Playbooks" escritos en formato YAML y no requiere un agente instalado en los nodos finales?
✓ Respuesta: C
Ansible es "agentless" y usa SSH para gestionar dispositivos mediante Playbooks en YAML.
Pregunta 20
¿Cuál es la función principal de Cisco DNA Center en una red empresarial?
✓ Respuesta: B
DNA Center es el controlador central para arquitecturas SD-Access, facilitando el diseño y la gestión de políticas.
Pregunta 50
¿Qué protocolo utiliza el puerto TCP 830 para la gestión de red programable?
✓ Respuesta: C
NETCONF utiliza el puerto 830 por defecto para establecer sesiones de gestión remota basadas en XML sobre SSH.
