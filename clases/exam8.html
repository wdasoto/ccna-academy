EXAMEN 8

IPv6, WAN & QoS

⏱️ Sin límite📝 30 preguntas
Pregunta 1
¿Cuál es el prefijo de una dirección IPv6 Link-Local?
✓ Respuesta: B
FE80::/10 son Link-Local (no enrutables, auto-generadas). 2001::/16=GUA, FC00::/7=ULA, FF00::/8=Multicast.
Pregunta 2
¿Qué protocolo reemplaza ARP en IPv6?
✓ Respuesta: A
NDP (ICMPv6 tipos 133-137) reemplaza ARP. Usa NS (Neighbor Solicitation) y NA (Neighbor Advertisement) para resolver IPv6→MAC.
Pregunta 3
¿Qué tecnología WAN usa células de 53 bytes?
✓ Respuesta: B
ATM usa células fijas de 53 bytes (5 header + 48 payload). Frame Relay usa frames variables. MPLS usa etiquetas.
Pregunta 4
¿Qué método de autoconfiguración IPv6 NO requiere DHCPv6?
✓ Respuesta: A
SLAAC usa RA (Router Advertisement) con prefijo + EUI-64 o privacy extensions. No necesita servidor DHCPv6.
Pregunta 5
¿Qué modelo de QoS clasifica y marca tráfico cerca del origen?
✓ Respuesta: C
DiffServ marca paquetes con DSCP (6 bits en ToS/TC), clasificación en edge, forwarding behavior en core. IntServ=reservas per-flow (no escalable).
Pregunta 6
¿Cuántos bits tiene una dirección IPv6?
✓ Respuesta: C
IPv6 = 128 bits (8 grupos de 16 bits hexadecimales). IPv4 = 32 bits.
Pregunta 7
¿Qué protocolo WAN usa etiquetas (labels) para conmutar paquetes?
✓ Respuesta: C
MPLS (Multiprotocol Label Switching) usa etiquetas de 20 bits para forwarding rápido (label swap en vez de lookup IP).
Pregunta 8
En IPv6, ¿qué dirección multicast alcanza todos los routers en el enlace local?
✓ Respuesta: B
FF02::2 = All-Routers. FF02::1 = All-Nodes. FF02::5 = OSPFv3. FF02::A = EIGRP.
Pregunta 9
¿Qué mecanismo de QoS descarta paquetes cuando la cola está llena?
✓ Respuesta: A
Tail Drop descarta todos los paquetes cuando cola=100%. WRED descarta probabilísticamente antes de llenarse (evita TCP global synchronization).
Pregunta 10
¿Qué tipo de VPN usa IPsec para túneles site-to-site?
✓ Respuesta: D
Todas usan IPsec para site-to-site. GRE over IPsec=túnel estático, DMVPN=túneles dinámicos hub-spoke, SSL VPN=clientless web (remote access).
Pregunta 11
¿Cuál es el tamaño de MTU mínimo en IPv6?
✓ Respuesta: B
IPv6 MTU mínimo = 1280 bytes (sin fragmentación). IPv4 mínimo = 576 bytes.
Pregunta 12
¿Qué comando habilita IPv6 routing en Cisco IOS?
✓ Respuesta: B
ipv6 unicast-routing habilita IPv6 routing globalmente. ipv6 enable solo genera link-local en interfaz.
Pregunta 13
¿Qué encabezado de extensión IPv6 se usa para fragmentación?
✓ Respuesta: B
Fragment Header maneja fragmentación (solo en origen, no en routers intermedios). Next Header = 44.
Pregunta 14
En QoS, ¿qué valor DSCP se recomienda para voz (VoIP)?
✓ Respuesta: A
EF (DSCP 46) para voz. AF41 (DSCP 34) para video. CS0 (DSCP 0) = best effort.
Pregunta 15
¿Qué tecnología WAN usa PPP sobre Ethernet?
✓ Respuesta: A
PPPoE (Point-to-Point Protocol over Ethernet) común en DSL. Usa discovery (PADI/PADO/PADR/PADS) + sesión PPP.
Pregunta 30
¿Qué técnica de QoS retrasa paquetes para suavizar ráfagas de tráfico?
✓ Respuesta: B
Shaping usa buffer para retrasar paquetes excedentes (suaviza ráfagas). Policing descarta/remarca inmediatamente.
