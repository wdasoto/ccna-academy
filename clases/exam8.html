EXAMEN 8

IPv6, WAN & QoS

⏱️ Sin límite📝 30 preguntas
Pregunta 1
¿Cuál es el prefijo de una dirección IPv6 Link-Local?
✓ Respuesta: B
FE80::/10 son Link-Local (no enrutables, auto-generadas). 2001::/16=GUA, FC00::/7=ULA, FF00::/8=Multicast.
Pregunta 2
¿Qué protocolo reemplaza ARP en IPv6?
✓ Respuesta: A
NDP (ICMPv6 tipos 133-137) reemplaza ARP. Usa NS (Neighbor Solicitation) y NA (Neighbor Advertisement) para resolver IPv6→MAC.
Pregunta 3
¿Qué tecnología WAN usa células de 53 bytes?
✓ Respuesta: B
ATM usa células fijas de 53 bytes (5 header + 48 payload). Frame Relay usa frames variables. MPLS usa etiquetas.
Pregunta 4
¿Qué método de autoconfiguración IPv6 NO requiere DHCPv6?
✓ Respuesta: A
SLAAC usa RA (Router Advertisement) con prefijo + EUI-64 o privacy extensions. No necesita servidor DHCPv6.
Pregunta 5
¿Qué modelo de QoS clasifica y marca tráfico cerca del origen?
✓ Respuesta: C
DiffServ marca paquetes con DSCP (6 bits en ToS/TC), clasificación en edge, forwarding behavior en core. IntServ=reservas per-flow (no escalable).
Pregunta 6
¿Cuántos bits tiene una dirección IPv6?
✓ Respuesta: C
IPv6 = 128 bits (8 grupos de 16 bits hexadecimales). IPv4 = 32 bits.
Pregunta 7
¿Qué protocolo WAN usa etiquetas (labels) para conmutar paquetes?
✓ Respuesta: C
MPLS (Multiprotocol Label Switching) usa etiquetas de 20 bits para forwarding rápido (label swap en vez de lookup IP).
Pregunta 8
En IPv6, ¿qué dirección multicast alcanza todos los routers en el enlace local?
✓ Respuesta: B
FF02::2 = All-Routers. FF02::1 = All-Nodes. FF02::5 = OSPFv3. FF02::A = EIGRP.
Pregunta 9
¿Qué mecanismo de QoS descarta paquetes cuando la cola está llena?
✓ Respuesta: A
Tail Drop descarta todos los paquetes cuando cola=100%. WRED descarta probabilísticamente antes de llenarse (evita TCP global synchronization).
Pregunta 10
¿Qué tipo de VPN usa IPsec para túneles site-to-site?
✓ Respuesta: D
Todas usan IPsec para site-to-site. GRE over IPsec=túnel estático, DMVPN=túneles dinámicos hub-spoke, SSL VPN=clientless web (remote access).
Pregunta 11
¿Cuál es el tamaño de MTU mínimo en IPv6?
✓ Respuesta: B
IPv6 MTU mínimo = 1280 bytes (sin fragmentación). IPv4 mínimo = 576 bytes.
Pregunta 12
¿Qué comando habilita IPv6 routing en Cisco IOS?
✓ Respuesta: B
ipv6 unicast-routing habilita IPv6 routing globalmente. ipv6 enable solo genera link-local en interfaz.
Pregunta 13
¿Qué encabezado de extensión IPv6 se usa para fragmentación?
✓ Respuesta: B
Fragment Header maneja fragmentación (solo en origen, no en routers intermedios). Next Header = 44.
Pregunta 14
En QoS, ¿qué valor DSCP se recomienda para voz (VoIP)?
✓ Respuesta: A
EF (DSCP 46) para voz. AF41 (DSCP 34) para video. CS0 (DSCP 0) = best effort.
Pregunta 15
¿Qué tecnología WAN usa PPP sobre Ethernet?
✓ Respuesta: A
PPPoE (Point-to-Point Protocol over Ethernet) común en DSL. Usa discovery (PADI/PADO/PADR/PADS) + sesión PPP.
Pregunta 16
¿Qué técnica de QoS retrasa paquetes para suavizar ráfagas de tráfico?
✓ Respuesta: B
Shaping usa buffer para retrasar paquetes excedentes (suaviza ráfagas). Policing descarda/remarca inmediatamente.
Pregunta 17
¿Cuál es el prefijo IPv6 utilizado exclusivamente para direcciones Link-Local?
✓ Respuesta: C
Las direcciones Link-Local (FE80::/10) se usan para la comunicación dentro del mismo segmento de enlace y no son enrutables fuera de él.
Pregunta 18
¿Qué campo del encabezado IPv6 se utiliza en QoS para marcar paquetes y es equivalente al campo ToS en IPv4?
✓ Respuesta: B
El campo Traffic Class de 8 bits en IPv6 cumple la misma función que el campo Type of Service (ToS) en IPv4 para la clasificación de tráfico.
Pregunta 19
¿Qué tecnología WAN utiliza etiquetas (labels) para reenviar paquetes en lugar de buscar en tablas de enrutamiento complejas?
✓ Respuesta: C
MPLS (Multi-Protocol Label Switching) inserta una etiqueta entre la capa 2 y la capa 3 para agilizar el reenvío de tráfico en redes de proveedores.
Pregunta 20
En QoS, ¿cuál es el término que describe la variación en el tiempo de llegada de los paquetes (retardo variable)?
✓ Respuesta: B
El Jitter es la fluctuación del retardo. Es especialmente crítico para aplicaciones de tiempo real como Voz sobre IP (VoIP).
Pregunta 21
Al configurar EUI-64 en IPv6, ¿qué valor se inserta en medio de la dirección MAC de 48 bits?
✓ Respuesta: C
Para EUI-64, se toma la MAC, se inserta FFFE en el medio y se invierte el séptimo bit (bit U/L).
Pregunta 22
¿Qué modelo de QoS proporciona garantías estrictas de ancho de banda mediante el uso de protocolos de señalización como RSVP?
✓ Respuesta: C
IntServ utiliza un enfoque de reserva de recursos de extremo a extremo, pero es difícil de escalar en redes grandes comparado con DiffServ.
Pregunta 23
¿Cuál es la dirección de Multicast de IPv6 que identifica a todos los routers en el segmento local?
✓ Respuesta: B
La dirección FF02::2 se refiere a todos los routers (All-Routers), mientras que FF02::1 se refiere a todos los nodos (All-Nodes).
Pregunta 24
¿Qué técnica de QoS descarta los paquetes excedentes de forma inmediata cuando el tráfico supera un límite definido?
✓ Respuesta: B
Traffic Policing descarta el tráfico sobrante (o lo remarca), provocando retransmisiones de TCP. Shaping, en cambio, los almacena en buffer.
Pregunta 25
¿Qué protocolo WAN de código abierto se utiliza frecuentemente para establecer túneles sobre Internet y soporta multicast y protocolos de enrutamiento?
✓ Respuesta: B
GRE es un protocolo de encapsulación que permite transportar una gran variedad de protocolos de red dentro de túneles IP punto a punto.
Pregunta 26
¿Cuál es el valor predeterminado del campo "Hop Limit" en un encabezado IPv6?
✓ Respuesta: C
Aunque puede variar según el Sistema Operativo, el estándar común para Hop Limit (equivalente al TTL de IPv4) en Cisco es 128 o 64. Para el examen CCNA se asocia frecuentemente a 64 o 128.
Pregunta 27
¿Qué mecanismo de QoS es responsable de decidir el orden en que los paquetes salen de una interfaz basándose en su prioridad?
✓ Respuesta: B
El Queuing (como CBWFQ o LLQ) gestiona el almacenamiento temporal y la programación de salida de los paquetes durante periodos de congestión.
Pregunta 28
¿Qué tipo de dirección IPv6 es ::1/128?
✓ Respuesta: B
La dirección ::1/128 es el equivalente IPv6 de la dirección 127.0.0.1 de IPv4 (Loopback).
Pregunta 29
¿Qué tecnología WAN permite la conectividad de sitios remotos a través de una arquitectura de software que separa el plano de control del plano de datos?
✓ Respuesta: C
SD-WAN (Software-Defined WAN) permite gestionar de forma centralizada la seguridad y el enrutamiento sobre múltiples tipos de conexiones físicas.
Pregunta 30
¿Qué valor de DSCP se recomienda típicamente para el tráfico de voz (VoIP)?
✓ Respuesta: C
El valor EF (DSCP 46) garantiza prioridad baja de retardo y pérdida para aplicaciones en tiempo real como la voz.
Pregunta 31
¿Qué protocolo utiliza IPv6 para reemplazar el protocolo ARP de IPv4?
✓ Respuesta: A
Neighbor Discovery usa mensajes de Neighbor Solicitation y Neighbor Advertisement para mapear direcciones IPv6 a direcciones MAC.
Pregunta 32
¿Cuál es la función del campo "Next Header" en el encabezado base de IPv6?
✓ Respuesta: B
El campo Next Header indica qué viene después del encabezado base, permitiendo una estructura modular mediante encabezados de extensión.
Pregunta 33
¿Qué tecnología WAN proporciona una conexión de fibra óptica de alta velocidad y es común en redes metropolitanas (MAN)?
✓ Respuesta: C
Metro Ethernet utiliza estándares de Ethernet sobre fibra para interconectar sedes en una misma área geográfica a velocidades de Gigabit.
Pregunta 34
¿Cuál es el propósito del mecanismo "Tail Drop" en QoS?
✓ Respuesta: B
Tail Drop es el comportamiento por defecto de una cola. El problema es que puede causar la sincronización global de TCP, por lo que se prefieren métodos como WRED.
Pregunta 35
¿Cómo se abrevia correctamente la dirección IPv6 2001:0DB8:0000:0000:0000:0000:1428:57ab?
✓ Respuesta: B
Se pueden eliminar los ceros a la izquierda y reemplazar grupos consecutivos de ceros por :: (una sola vez por dirección).
Pregunta 36
¿Qué tipo de encapsulación es la predeterminada en las interfaces seriales de los routers Cisco?
✓ Respuesta: B
Cisco utiliza su propia versión de HDLC como encapsulación por defecto en enlaces seriales síncronos.
Pregunta 37
¿Qué herramienta de QoS utiliza un "Token Bucket" para suavizar las ráfagas de tráfico enviando los excedentes a una cola de espera?
✓ Respuesta: B
El Traffic Shaping retiene los paquetes en una cola cuando se excede la tasa configurada, resultando en un flujo de salida más constante y ordenado.
Pregunta 38
¿Cuál es el rango de direcciones IPv6 de Multicast?
✓ Respuesta: B
Todas las direcciones de Multicast en IPv6 comienzan con el primer octeto en FF (FF00::/8).
Pregunta 39
¿Qué campo del encabezado de trama 802.1Q se utiliza para marcar la prioridad de Capa 2 (CoS)?
✓ Respuesta: C
En el Tag de VLAN de 802.1Q existen 3 bits llamados Priority (User Priority) que definen el Class of Service (CoS).
Pregunta 40
¿Qué protocolo de autenticación PPP envía las credenciales en texto plano (sin cifrar)?
✓ Respuesta: B
PAP (Password Authentication Protocol) no es seguro porque las contraseñas viajan en texto claro. CHAP es la alternativa segura que usa hash.
Pregunta 41
¿Qué permite el comando "ipv6 unicast-routing" en un router Cisco?
✓ Respuesta: B
Sin el comando ipv6 unicast-routing, un router Cisco se comporta como un host IPv6 y no procesará rutas.
Pregunta 42
¿Qué mecanismo de QoS ayuda a prevenir la congestión descartando paquetes de baja prioridad de forma aleatoria antes de que la cola se llene?
✓ Respuesta: B
WRED evita que las colas se llenen por completo y mitiga la sincronización de ventanas de TCP descartando paquetes antes de que ocurra una congestión crítica.
Pregunta 43
¿Cuál es la longitud total de una dirección IPv6 en bits?
✓ Respuesta: C
IPv6 utiliza un espacio de direcciones de 128 bits, permitiendo un número casi ilimitado de direcciones únicas.
Pregunta 44
¿En qué consiste el método de transición "Dual Stack"?
✓ Respuesta: C
Dual Stack permite que los dispositivos hablen ambos protocolos al mismo tiempo durante el periodo de transición hacia IPv6 puro.
Pregunta 45
¿Qué tipo de VPN WAN se basa en el intercambio de etiquetas por parte del Service Provider para separar el tráfico de diferentes clientes?
✓ Respuesta: B
En las MPLS VPN de Capa 3, el proveedor utiliza VRFs y etiquetas MPLS para asegurar que el tráfico de una empresa no sea visible para otra.
Pregunta 46
¿Cuál es el prefijo para las direcciones IPv6 de tipo Unique Local?
✓ Respuesta: C
Las direcciones Unique Local (ULA) son el equivalente a las IPs privadas de IPv4 (RFC 1918).
Pregunta 47
¿Qué componente de QoS permite identificar aplicaciones específicas (como Facebook o YouTube) analizando la carga útil del paquete más allá del puerto TCP/UDP?
✓ Respuesta: B
NBAR realiza una inspección profunda de paquetes (DPI) para clasificar tráfico que usa puertos dinámicos o que intenta esconderse en puertos comunes como el 443.
Pregunta 48
¿Qué método permite a un host IPv6 obtener su dirección IP, prefijo y gateway sin necesidad de un servidor DHCPv6?
✓ Respuesta: B
SLAAC utiliza mensajes de ICMPv6 Router Advertisement para que los hosts generen sus propias direcciones IP basándose en el prefijo anunciado por el router.
Pregunta 49
¿Cuál es el retardo máximo recomendado (one-way latency) para el tráfico de voz para mantener una buena calidad de llamada?
✓ Respuesta: B
Cisco y la industria recomiendan un retardo de extremo a extremo no mayor a 150 ms para garantizar una experiencia de voz fluida.
Pregunta 50
¿Qué tecnología WAN utiliza una topología Hub-and-Spoke para construir una red VPN multipunto dinámica sobre Internet pública?
✓ Respuesta: B
DMVPN permite conectar múltiples sitios de forma segura simplificando la configuración de túneles IPsec mediante el uso de mGRE y NHRP.
