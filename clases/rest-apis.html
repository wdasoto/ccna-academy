🎯
Lo que entra en el examen CCNA sobre REST APIs

El examen CCNA 200-301 exige conocer las características de las APIs REST (Domain 6.5): verbos HTTP CRUD, codificación de datos, códigos de estado, y la diferencia entre RESTCONF y NETCONF. También evalúa arquitecturas SDN, SD-Access y el rol del controlador (6.3, 6.4). Las preguntas son conceptuales: identificar qué verbo HTTP usar para cada operación CRUD.
