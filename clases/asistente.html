🔑 CONFIGURAR API KEY

NetBot usa la API de Anthropic (Claude) para generar configuraciones. Necesitas una API key gratuita de console.anthropic.com

Cómo obtenerla:
1. Ve a console.anthropic.com
2. Crea cuenta gratuita
3. Settings → API Keys → Create Key
4. Copia la key (empieza con sk-ant-)

🔒 La key se guarda solo en tu navegador (localStorage). Nunca se envía a terceros.
