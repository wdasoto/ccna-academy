EXAMEN 7

Seguridad de Redes (Port Security, AAA, VPN, Wireless)

⏱️ Sin límite📝 30 preguntas
Pregunta 1
¿Qué acción de Port Security desactiva el puerto y envía trap SNMP?
✓ Respuesta: C
shutdown (default) pone puerto en err-disabled, incrementa contador violaciones, envía trap SNMP y syslog. restrict=descarta frames + log. protect=solo descarta frames.
Pregunta 2
¿Qué protocolo AAA usa TCP y cifra todo el paquete?
✓ Respuesta: B
TACACS+: TCP/49, cifra payload completo, separa AAA. RADIUS: UDP/1812-1813, solo cifra password, combina authn/authz.
Pregunta 3
¿Qué ataque mitiga DHCP Snooping?
✓ Respuesta: B
DHCP Snooping crea binding table (MAC-IP-VLAN-port), marca puertos trusted/untrusted, bloquea DHCPOFFER/ACK en untrusted. Mitiga rogue servers y DHCP starvation.
Pregunta 4
¿Qué característica impide CAM table overflow?
✓ Respuesta: A
Port Security limita MACs por puerto. CAM overflow (macof attack) llena tabla con MACs falsas → switch actúa como hub. Port sec con switchport port-security maximum 2 mitiga esto.
Pregunta 5
¿Qué protocolo de autenticación wireless usa servidor RADIUS?
✓ Respuesta: B
WPA2-Enterprise usa 802.1X (EAP) con servidor RADIUS para autenticación centralizada. WPA2-Personal usa PSK (pre-shared key) sin servidor.
Pregunta 6
¿Cuál es el estado predeterminado de una interfaz de switch cuando ocurre una violación de Port Security y no se ha configurado un modo específico?
✓ Respuesta: C
El modo Shutdown es el predeterminado. La interfaz se pone en estado error-disabled, se incrementa el contador de violaciones y se envía un mensaje SNMP/Syslog.
Pregunta 7
¿Qué componente del modelo AAA se encarga de determinar a qué recursos y comandos tiene permitido acceder un usuario?
✓ Respuesta: B
La Autorización define los privilegios del usuario (qué puede hacer), mientras que la Autenticación verifica quién es y el Accounting registra qué hizo.
Pregunta 8
¿Qué tipo de VPN utiliza el protocolo TLS/SSL para proporcionar acceso remoto seguro sin necesidad de instalar un software cliente especializado?
✓ Respuesta: C
Una Clientless SSL VPN permite a los usuarios acceder a recursos corporativos usando solo un navegador web estándar mediante cifrado SSL/TLS.
Pregunta 9
En Port Security, ¿qué comando permite que las direcciones MAC aprendidas dinámicamente se guarden en la configuración persistente (running-config)?
✓ Respuesta: B
La opción sticky convierte las direcciones dinámicas en "pegajosas", permitiendo que se almacenen en la configuración y sobrevivan a un reinicio si se guarda la configuración.
Pregunta 10
¿Cuál es el protocolo propietario de Cisco utilizado para la comunicación entre un NAS y un servidor AAA que cifra todo el cuerpo del paquete?
✓ Respuesta: C
TACACS+ cifra todo el contenido del paquete (a diferencia de RADIUS que solo cifra el password) y separa las funciones de A, A y A.
Pregunta 11
¿Qué algoritmo de cifrado es el estándar para WPA2?
✓ Respuesta: C
AES (Advanced Encryption Standard) con el protocolo CCMP es el estándar de cifrado robusto para WPA2.
Pregunta 12
¿Qué modo de violación de Port Security descarta el tráfico no autorizado y genera un mensaje de log, pero mantiene la interfaz activa?
✓ Respuesta: B
El modo Restrict descarta el tráfico, incrementa el contador de violaciones y envía notificaciones, sin apagar la interfaz. El modo Protect hace lo mismo pero sin notificaciones.
Pregunta 13
¿Cuál es el propósito principal de un SSID en una red inalámbrica?
✓ Respuesta: B
El SSID (Service Set Identifier) es el nombre de la red inalámbrica que los usuarios ven al intentar conectarse.
Pregunta 14
¿Qué mecanismo de seguridad en switches previene que un atacante suplante a un servidor DHCP legítimo?
✓ Respuesta: C
DHCP Snooping clasifica interfaces como "trusted" (donde está el servidor real) y "untrusted", bloqueando mensajes DHCP Offer de puertos no autorizados.
Pregunta 15
¿Qué protocolo utiliza una VPN Site-to-Site para autenticar, cifrar y verificar la integridad de los paquetes a nivel de capa 3?
✓ Respuesta: B
IPsec (Internet Protocol Security) es el conjunto de protocolos estándar para asegurar comunicaciones IP mediante autenticación y cifrado.
Pregunta 16
¿Qué modo de operación inalámbrica permite que los APs sean gestionados de forma centralizada por un WLC?
✓ Respuesta: B
Los Lightweight APs funcionan con el protocolo CAPWAP para comunicarse con un controlador (WLC), el cual gestiona la configuración y políticas.
Pregunta 17
¿Cuál es la función del protocolo CAPWAP en una arquitectura WLAN de Cisco?
✓ Respuesta: B
CAPWAP crea túneles para transportar tanto el tráfico de gestión (Control) como el tráfico de los clientes (Data) hacia el controlador centralizado.
Pregunta 18
¿Qué comando global activa el modelo AAA en un dispositivo Cisco?
✓ Respuesta: C
El comando aaa new-model es indispensable para habilitar el conjunto de comandos AAA en el router o switch.
Pregunta 19
¿Qué característica de seguridad inalámbrica requiere que el cliente ingrese una contraseña compartida antes de asociarse, pero no ofrece autenticación individualizada?
✓ Respuesta: C
En WPA2-PSK todos los usuarios comparten la misma clave (Pre-Shared Key), lo cual es ideal para hogares pero menos seguro para empresas.
Pregunta 20
¿Cuál es la función del plano de control en una red inalámbrica con controlador (WLC)?
✓ Respuesta: B
El Plano de Control en arquitecturas centralizadas se encarga de las decisiones inteligentes como el roaming, la potencia de radio y la autenticación.
Pregunta 21
¿Qué técnica de ataque intenta desbordar la tabla de direcciones MAC de un switch para forzarlo a actuar como un hub?
✓ Respuesta: C
El MAC Address Flooding inunda la tabla CAM con direcciones falsas. Port Security es la defensa principal contra esto.
Pregunta 22
¿Qué tipo de amenaza inalámbrica consiste en un AP instalado en la red sin autorización del departamento de IT?
✓ Respuesta: B
Un Rogue AP es un punto de acceso no autorizado conectado físicamente a la red cableada de la empresa, creando un agujero de seguridad.
Pregunta 23
¿Cuál es la principal ventaja de usar WPA3 sobre WPA2?
✓ Respuesta: B
WPA3 introduce SAE (Simultaneous Authentication of Equals), que reemplaza el handshake de WPA2 y protege contra ataques de diccionario fuera de línea.
Pregunta 24
¿Qué puerto UDP utiliza TACACS+ para su funcionamiento?
✓ Respuesta: B
A diferencia de RADIUS (que usa UDP), TACACS+ utiliza el protocolo de transporte **TCP** en el puerto **49**.
Pregunta 25
¿Qué tecnología VPN permite crear túneles dinámicos multipunto bajo demanda (Spoke-to-Spoke) sin intervención manual constante?
✓ Respuesta: C
DMVPN combina multipoint GRE, NHRP e IPsec para permitir que las sucursales hablen entre sí directamente sin pasar por el Hub.
Pregunta 26
¿En qué consiste el "Dynamic ARP Inspection" (DAI)?
✓ Respuesta: B
DAI previene ataques de ARP Poisoning interceptando y validando cada respuesta ARP antes de actualizar la tabla local.
Pregunta 27
¿Qué banda de frecuencia ofrece más canales no superpuestos para redes Wireless?
✓ Respuesta: B
La banda de 5 GHz tiene muchos más canales (hasta 24 o más dependiendo del país) que no se solapan, comparada con los 3 canales de la banda de 2.4 GHz (1, 6, 11).
Pregunta 28
¿Cuál es el propósito del protocolo NHRP en DMVPN?
✓ Respuesta: B
NHRP (Next Hop Resolution Protocol) actúa como un "servicio de directorio" donde los Spokes registran sus IPs públicas para que el Hub y otros Spokes sepan dónde encontrarlos.
Pregunta 29
¿Qué característica de seguridad desactiva automáticamente un puerto de switch si recibe una unidad de datos de protocolo de puente (BPDU) no esperada?
✓ Respuesta: C
BPDU Guard se usa en puertos de acceso (PortFast). Si alguien conecta un switch a ese puerto, la interfaz se apaga para prevenir bucles de capa 2 y cambios en la topología STP.
Pregunta 30
¿Qué protocolo EAP es considerado uno de los más seguros porque requiere certificados digitales tanto en el servidor como en el cliente?
✓ Respuesta: A
EAP-TLS requiere una infraestructura de clave pública (PKI) y certificados en ambos extremos, ofreciendo el nivel más alto de seguridad inalámbrica.
Pregunta 31
¿Cuál es el comando para ver el estado de las interfaces puestas en "error-disabled" por seguridad?
✓ Respuesta: A
El comando show interfaces status muestra claramente el estado "err-disabled" en la columna de Status de la interfaz afectada.
Pregunta 32
¿Qué ataque de red utiliza mensajes ICMP Redirect o falsas respuestas ARP para interceptar tráfico entre dos dispositivos?
✓ Respuesta: B
El ataque Man-in-the-Middle busca situar al atacante en la ruta de comunicación para espiar o modificar los datos sin que las víctimas lo noten.
Pregunta 33
¿Cuál es la función de una VPN "Split-Tunneling"?
✓ Respuesta: B
El Split-Tunneling optimiza el ancho de banda al evitar que el tráfico no esencial (como ver YouTube) pase por la red privada de la empresa.
Pregunta 34
¿Qué protocolo inalámbrico es vulnerable al ataque "WPS brute force"?
✓ Respuesta: C
WPS tiene una debilidad de diseño en su PIN de 8 dígitos que permite a atacantes obtener la clave PSK en pocas horas. Se recomienda desactivarlo.
Pregunta 35
¿Qué componente del túnel IPsec se encarga de negociar los algoritmos de seguridad (Security Associations)?
✓ Respuesta: C
IKE es el protocolo que gestiona la autenticación de los peers y la negociación de los parámetros de cifrado en las Fases 1 y 2.
Pregunta 36
¿Qué comando se usa para habilitar Port Security en una interfaz de switch específica?
✓ Respuesta: C
El comando switchport port-security (sin parámetros adicionales) es el que activa la funcionalidad en la interfaz (previamente configurada como modo acceso).
Pregunta 37
¿Cuál es el beneficio de SNMPv3 sobre sus versiones anteriores (v1 y v2c)?
✓ Respuesta: B
SNMPv3 introduce seguridad criptográfica, evitando que las "community strings" viajen en texto plano como ocurre en v1 y v2c.
Pregunta 38
¿Qué tipo de VPN es más adecuada para conectar permanentemente dos oficinas corporativas a través de Internet?
✓ Respuesta: B
La VPN Site-to-Site une dos redes completas (LAN a LAN) de forma transparente para los usuarios finales.
Pregunta 39
¿Cuál es la función del servidor RADIUS en una autenticación 802.1X?
✓ Respuesta: C
En el estándar 802.1X, el **Supplicant** es el cliente, el **Authenticator** es el switch o AP, y el **Authentication Server** es el servidor AAA (RADIUS).
Pregunta 40
¿Qué algoritmo se usa en el proceso de intercambio de llaves IPsec para que dos dispositivos generen una llave secreta compartida sobre un canal inseguro?
✓ Respuesta: C
Diffie-Hellman es el método matemático que permite establecer una clave compartida sin haberla enviado nunca a través de la red.
Pregunta 41
¿Qué vulnerabilidad de seguridad de red permite a un atacante enviar paquetes IP con una dirección de origen falsa?
✓ Respuesta: A
El IP Spoofing se usa a menudo en ataques DoS para ocultar la identidad del atacante o engañar a sistemas basados en confianza por IP.
Pregunta 42
¿Qué modo de WPA2 utiliza una llave de cifrado diferente para cada usuario basada en su login personal?
✓ Respuesta: B
WPA2-Enterprise genera dinámicamente llaves únicas para cada sesión de usuario tras una autenticación exitosa contra un servidor RADIUS.
Pregunta 43
¿Qué protocolo IPsec asegura la integridad de los datos y el origen del mensaje, pero NO proporciona cifrado (confidencialidad)?
✓ Respuesta: B
AH solo garantiza que el paquete no fue modificado y que viene de quien dice venir. Si se requiere cifrado, se debe usar ESP.
Pregunta 44
¿Cuál es la función del plano de datos (Data Plane) en un AP autónomo?
✓ Respuesta: B
En un AP Autónomo, el propio AP gestiona todo: control y datos. El plano de datos es el encargado del movimiento real de los frames inalámbricos a la red cableada.
Pregunta 45
¿Cómo se le llama a un ataque MitM donde el atacante crea un punto de acceso inalámbrico con el mismo SSID que una red legítima?
✓ Respuesta: C
Un Evil Twin engaña a los dispositivos de los usuarios para que se conecten al AP del atacante creyendo que es la red oficial, permitiendo capturar datos sensibles.
Pregunta 46
¿Qué comando permite habilitar el registro de los comandos ejecutados por un usuario en un servidor AAA (Accounting)?
✓ Respuesta: A
Este comando envía información al servidor TACACS+ cada vez que un usuario con nivel de privilegio 15 inicia o termina la ejecución de un comando.
Pregunta 47
¿Cuál es el valor máximo de direcciones MAC que se pueden configurar por puerto mediante Port Security?
✓ Respuesta: D
Aunque el valor por defecto es 1, el límite máximo permitido varía según las capacidades de hardware de cada modelo específico de switch Cisco.
Pregunta 48
¿Qué estándar inalámbrico introdujo el soporte para bandas de 2.4 GHz y 5 GHz simultáneamente y alcanzó velocidades de hasta 600 Mbps?
✓ Respuesta: C
802.11n fue el primer estándar en usar MIMO (Multiple Input Multiple Output) y operar en ambas bandas de frecuencia.
Pregunta 49
¿Qué modo de Wireless LAN Controller (WLC) se utiliza para instalaciones pequeñas donde el controlador está integrado físicamente en un switch Catalyst?
✓ Respuesta: B
El Embedded WLC corre directamente sobre el hardware de switches específicos (como la serie Catalyst 9000), ahorrando espacio y costos de hardware adicional.
Pregunta 50
¿Cuál es la función principal del protocolo Cisco Management Frame Protection (MFP)?
✓ Respuesta: B
Cisco MFP añade una firma criptográfica a las tramas de gestión (como las de desasociación). Si un atacante intenta enviar una trama falsa para desconectar a los usuarios, el AP o el cliente detectarán que la firma no es válida y la ignorarán.
