Backup Automático

Script completo para hacer backup automático de running-config de múltiples dispositivos Cisco. Organización por fecha y hostname en archivos locales, gestión de inventario CSV y programación de tareas. Protege tu infraestructura con copias de seguridad programadas.

💾 Backup Masivo 🗄️ Archivos .txt 📅 Datetime 📊 Inventario CSV ⏰ Automatización

¿Por qué automatizar backups?

El running-config de tus dispositivos es el activo más valioso de tu red. Una configuración incorrecta, un fallo de hardware o un ataque pueden dejar tu infraestructura inoperativa. Los backups manuales son propensos al olvido; la automatización garantiza que siempre tengas una copia reciente de seguridad.

En este tema construiremos un sistema completo: lectura de inventario desde CSV, conexión SSH masiva, almacenamiento organizado por fechas y automatización con el programador de tareas del sistema operativo.

🛡️

Protección ante Desastres

Recuperación rápida ante fallos. Un backup reciente puede reducir el MTTR (Mean Time To Repair) de horas a minutos.

⚡

Velocidad y Escala

Realizar backup de 100 routers manualmente toma días. Con Python, toma minutos y se ejecuta mientras duermes.

📅

Historial de Cambios

Al organizar por fechas, tienes un versionado natural. Puedes comparar qué cambió entre ayer y hoy.

✅

Cumplimiento

Auditorías y estándares (ISO 27001, SOC 2) requieren backups periódicos y documentados de configuraciones.

💡 Componentes del Sistema
  • os & datetime: Crear carpetas y timestamps
  • csv: Leer inventario de dispositivos
  • Netmiko: Conectar y extraer configuración
  • logging: Registrar éxitos y fallos
  • schedule/cron: Ejecutar automáticamente

Conexión segura para backup

Usaremos Netmiko para conectar, pero agregaremos manejo de errores específico para backups: timeout extendido (las configs grandes tardan) y verificación de enable mode.

conexion_backup.py — Función robusta de conexión
from netmiko import ConnectHandler, NetmikoTimeoutException, NetmikoAuthenticationException
import logging

# Configurar logging para ver qué pasó
logging.basicConfig(
    filename="backup.log",
    level=logging.INFO,
    format="%(asctime)s - %(levelname)s - %(message)s"
)

def conectar_para_backup(device_info):
    """
    Establece conexión SSH optimizada para descarga de config.
    Retorna objeto conexión o None si falla.
    """
    # Añadir parámetros específicos para backup
    device_info["device_type"] = "cisco_ios"
    device_info["timeout"] = 60       # Más tiempo para configs grandes
    device_info["session_timeout"] = 120
    
    try:
        conn = ConnectHandler(**device_info)
        # Entrar a enable mode si es necesario
        if "#" not in conn.find_prompt():
            conn.enable()
        
        logging.info(f"Conectado a {device_info['ip']}")
        return conn
        
    except NetmikoAuthenticationException:
        logging.error(f"Auth fallido en {device_info['ip']}")
        return None
    except NetmikoTimeoutException:
        logging.error(f"Timeout en {device_info['ip']}")
        return None
    except Exception as e:
        logging.error(f"Error en {device_info['ip']}: {e}")
        return None
⚠️ Seguridad en backups

Los archivos de backup contienen contraseñas en texto plano (enable secrets, SNMP communities). Almacénalos en:

  • Carpeta con permisos restringidos (chmod 700)
  • Unidad encriptada (BitLocker, LUKS)
  • Repositorio privado (nunca en Git público)

show running-config

El comando más importante para backup. Algunos dispositivos tienen configuraciones tan grandes que requieren paginación controlada.

descargar_config.py — Obtener y limpiar
def obtener_running_config(conn, hostname="unknown"):
    """
    Ejecuta show run y limpia el output para guardar.
    """
    print(f"📥 Descargando config de {hostname}...")
    
    # Comando con delay extendido para dispositivos lentos
    output = conn.send_command(
        "show running-config",
        delay_factor=4,           # Espera más si hay mucho output
        strip_prompt=True,       # Quitar "Router#>" del final
        strip_command=True       # Quitar "show running-config" del inicio
    )
    
    # Limpiar líneas de "Building configuration..."
    lines = output.split("\n")
    clean_lines = []
    for line in lines:
        if "Building configuration" in line:
            continue
        if "Current configuration" in line and "bytes" in line:
            continue
        clean_lines.append(line)
    
    return "\n".join(clean_lines)

# Uso
with ConnectHandler(**device) as conn:
    config = obtener_running_config(conn, "R1-CORE")
    print(f"Descargados {len(config)} caracteres")

Backup de startup-config también

dual_backup.py — Running + Startup
def backup_completo(conn, hostname, backup_dir):
    """Guarda running y startup"""
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    
    # Obtener ambas configuraciones
    running = conn.send_command("show running-config")
    startup = conn.send_command("show startup-config")
    
    # Guardar running
    with open(f"{backup_dir}/{hostname}_running_{timestamp}.cfg", "w") as f:
        f.write(running)
    
    # Guardar startup
    with open(f"{backup_dir}/{hostname}_startup_{timestamp}.cfg", "w") as f:
        f.write(startup)
    
    # Verificar si hay diferencias (drift)
    if running != startup:
        logging.warning(f"{hostname}: Running y Startup difieren (config no guardada)")

import os
from pathlib import Path

def crear_estructura_backup(base_dir="backups"):
    """Crea árbol de carpetas: backups/YYYY/MM/DD/"""
    hoy = datetime.now()
    year = hoy.strftime("%Y")
    month = hoy.strftime("%m")
    day = hoy.strftime("%d")
    
    path = Path(base_dir) / year / month / day
    path.mkdir(parents=True, exist_ok=True)
    
    return str(path)

import csv
import ipaddress

def cargar_inventario_validado(archivo="inventario.csv"):
    """Carga y valida cada entrada del CSV"""
    dispositivos = []
    errores = []
    
    with open(archivo, "r") as f:
        reader = csv.DictReader(f)
        for i, row in enumerate(reader, start=2):
            # Validar campos obligatorios
            if not all([row.get("ip"), row.get("hostname")]):
                errores.append(f"Línea {i}: Faltan campos obligatorios")
                continue
            
            # Validar IP
            try:
                ipaddress.ip_address(row["ip"])
            except ValueError:
                errores.append(f"Línea {i}: IP inválida {row['ip']}")
                continue
            
            dispositivos.append(row)
    
    if errores:
        print("⚠️ Errores encontrados:")
        for e in errores:
            print(f"  {e}")
    
    return dispositivos

Script de backup masivo

Integración completa: lee CSV, conecta a cada uno, guarda con timestamp y genera reporte.

backup_masivo.py — Script completo
import csv
import os
from datetime import datetime
from netmiko import ConnectHandler
from netmiko.exceptions import NetmikoTimeoutException, NetmikoAuthenticationException

def backup_dispositivo(device, backup_dir):
    """Realiza backup de un solo dispositivo"""
    print(f"\n🔌 Conectando a {device['hostname']} ({device['ip']})...")
    
    conn_dict = {
        "device_type": device.get("device_type", "cisco_ios"),
        "host": device["ip"],
        "username": device["username"],
        "password": device["password"],
        "secret": device.get("enable_secret", ""),
    }
    
    try:
        with ConnectHandler(**conn_dict) as conn:
            # Entrar a enable si es necesario
            if conn_dict["secret"]:
                conn.enable()
            
            # Obtener config
            output = conn.send_command("show running-config", delay_factor=2)
            
            # Crear nombre de archivo
            timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
            filename = f"{backup_dir}/{device['hostname']}_{timestamp}.txt"
            
            # Guardar
            with open(filename, "w") as f:
                f.write(f"! Backup de {device['hostname']}\n")
                f.write(f"! Fecha: {timestamp}\n!\n")
                f.write(output)
            
            print(f"   ✅ Guardado: {filename}")
            return {"hostname": device["hostname"], "status": "success", "file": filename}
            
    except Exception as e:
        print(f"   ❌ Error: {e}")
        return {"hostname": device["hostname"], "status": "error", "error": str(e)}

# === EJECUCIÓN PRINCIPAL ===
if __name__ == "__main__":
    # Crear carpeta de hoy
    hoy = datetime.now().strftime("%Y%m%d")
    backup_dir = f"backups/{hoy}"
    os.makedirs(backup_dir, exist_ok=True)
    
    # Cargar inventario
    devices = []
    with open("inventario.csv") as f:
        devices = list(csv.DictReader(f))
    
    print(f"🚀 Iniciando backup de {len(devices)} dispositivos...")
    print(f"📁 Destino: {backup_dir}/")
    
    # Ejecutar backups
    resultados = []
    for dev in devices:
        resultados.append(backup_dispositivo(dev, backup_dir))
    
    # Reporte final
    exitosos = [r for r in resultados if r["status"] == "success"]
    print(f"\n🏁 Completado: {len(exitosos)}/{len(devices)} exitosos")

💡 Buenas prácticas de scheduling
  • Horario bajo: 2:00-4:00 AM cuando la red está quieta
  • Rotación: Mantener solo últimos 30 días (script de limpieza)
  • Notificaciones: Enviar email si hay errores (usar smtplib)
  • Lock file: Evitar ejecuciones simultáneas si el anterior tarda

🚀 Proyecto: Sistema Enterprise de Backup

Script profesional con todas las características: validación, logging, reporte HTML y limpieza de archivos antiguos.

backup_enterprise.py — Versión producción
import csv
import os
import shutil
from datetime import datetime, timedelta
from pathlib import Path
from netmiko import ConnectHandler
from netmiko.exceptions import NetmikoTimeoutException, NetmikoAuthenticationException
import logging
import smtplib
from email.mime.text import MIMEText

# Configuración
CONFIG = {
    "inventario": "inventario.csv",
    "backup_base": "backups",
    "retencion_dias": 30,
    "email_errores": "admin@empresa.com",
}

def setup_logging():
    """Configura logging con rotación diaria"""
    log_dir = "logs"
    os.makedirs(log_dir, exist_ok=True)
    fecha = datetime.now().strftime("%Y%m%d")
    logging.basicConfig(
        filename=f"{log_dir}/backup_{fecha}.log",
        level=logging.INFO,
        format="%(asctime)s [%(levelname)s] %(message)s"
    )

def limpiar_antiguos(backup_base, dias_retencion):
    """Elimina carpetas de backup más antiguas que X días"""
    limite = datetime.now() - timedelta(days=dias_retencion)
    for carpeta in Path(backup_base).glob("*"):
        if carpeta.is_dir():
            try:
                fecha_carpeta = datetime.strptime(carpeta.name, "%Y%m%d")
                if fecha_carpeta < limite:
                    shutil.rmtree(carpeta)
                    logging.info(f"Eliminada carpeta antigua: {carpeta}")
            except ValueError:
                pass  # No es formato de fecha

def backup_dispositivo(device, backup_dir):
    """Realiza backup con manejo completo de errores"""
    hostname = device["hostname"]
    ip = device["ip"]
    
    device_params = {
        "device_type": device.get("device_type", "cisco_ios"),
        "host": ip,
        "username": device["username"],
        "password": device["password"],
        "secret": device.get("enable_secret", ""),
        "timeout": 60,
    }
    
    try:
        with ConnectHandler(**device_params) as conn:
            if device_params["secret"]:
                conn.enable()
            
            # Obtener configuración
            config = conn.send_command("show running-config", delay_factor=3)
            
            # Crear archivo con metadata
            timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
            filename = f"{backup_dir}/{hostname}_{timestamp}.cfg"
            
            with open(filename, "w") as f:
                f.write(f"! Backup generado: {timestamp}\n")
                f.write(f"! Hostname: {hostname}\n")
                f.write(f"! IP: {ip}\n")
                f.write("!" + "="*50 + "\n")
                f.write(config)
            
            logging.info(f"✅ {hostname}: Backup exitoso ({len(config)} chars)")
            return {"status": "success", "hostname": hostname, "file": filename}
            
    except NetmikoAuthenticationException as e:
        logging.error(f"❌ {hostname}: Autenticación fallida - {e}")
        return {"status": "error", "hostname": hostname, "error": "Auth failed"}
    except NetmikoTimeoutException as e:
        logging.error(f"❌ {hostname}: Timeout - {e}")
        return {"status": "error", "hostname": hostname, "error": "Timeout"}
    except Exception as e:
        logging.error(f"❌ {hostname}: Error general - {e}")
        return {"status": "error", "hostname": hostname, "error": str(e)}

def generar_reporte_html(resultados, backup_dir):
    """Genera reporte visual del backup"""
    exitosos = [r for r in resultados if r["status"] == "success"]
    fallidos = [r for r in resultados if r["status"] == "error"]
    
    html = f"""
<!DOCTYPE html>
<html>
<head>
    <title>Reporte Backup {datetime.now().strftime('%Y-%m-%d')}</title>
    <style>
        body {{ font-family: Arial; margin: 20px; }}
        .success {{ color: green; }}
        .error {{ color: red; }}
        table {{ border-collapse: collapse; width: 100%; }}
        th, td {{ border: 1px solid #ddd; padding: 8px; text-align: left; }}
        th {{ background-color: #f2f2f2; }}
    </style>
</head>
<body>
    <h1>Reporte de Backup - {datetime.now().strftime('%Y-%m-%d %H:%M')}</h1>
    <p>Exitosos: {len(exitosos)} | Fallidos: {len(fallidos)}</p>
    
    <h2>Resumen</h2>
    <table>
        <tr><th>Hostname</th><th>Estado</th><th>Detalle</th></tr>
        {''.join(f"<tr><td>{r['hostname']}</td><td class='{r['status']}'>{r['status']}</td><td>{r.get('file', r.get('error', ''))}</td></tr>" for r in resultados)}
    </table>
</body>
</html>
"""
    
    reporte_file = f"{backup_dir}/reporte.html"
    with open(reporte_file, "w") as f:
        f.write(html)
    return reporte_file

# === EJECUCIÓN PRINCIPAL ===
if __name__ == "__main__":
    setup_logging()
    logging.info("=== INICIANDO BACKUP ENTERPRISE ===")
    
    # Crear estructura de carpetas
    hoy = datetime.now().strftime("%Y%m%d")
    backup_dir = f"{CONFIG['backup_base']}/{hoy}"
    os.makedirs(backup_dir, exist_ok=True)
    
    # Limpiar archivos antiguos
    limpiar_antiguos(CONFIG["backup_base"], CONFIG["retencion_dias"])
    
    # Cargar inventario
    devices = []
    try:
        with open(CONFIG["inventario"]) as f:
            devices = list(csv.DictReader(f))
    except FileNotFoundError:
        logging.error(f"Inventario no encontrado: {CONFIG['inventario']}")
        exit(1)
    
    # Ejecutar backups
    resultados = []
    for device in devices:
        resultado = backup_dispositivo(device, backup_dir)
        resultados.append(resultado)
    
    # Generar reporte
    reporte = generar_reporte_html(resultados, backup_dir)
    
    # Resumen final
    exitosos = len([r for r in resultados if r["status"] == "success"])
    logging.info(f"=== BACKUP COMPLETADO: {exitosos}/{len(devices)} exitosos ===")
    print(f"\n🏁 Reporte generado: {reporte}")
