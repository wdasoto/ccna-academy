¿Cómo agregar una nueva clase?

Para agregar una nueva clase al portal, sigue estos pasos:

  1. Crea el archivo HTML de la clase en la carpeta clases/ (ej: clases/ospf.html)
  2. Copia la plantilla clases/_plantilla.html como punto de partida
  3. En index.html, cambia el class="lesson-card coming" a class="lesson-card active" y agrega el enlace href
  4. Actualiza el contador de progreso del módulo correspondiente
  5. Haz commit y push a GitHub — GitHub Pages actualiza automáticamente
