Lo que entra en el examen CCNA sobre JSON

El examen CCNA 200-301 exige reconocer componentes de datos JSON (Domain 6.7). Debes identificar: objetos vs arrays, tipos de datos, anidamiento, y la diferencia entre JSON (RESTCONF), XML (NETCONF) y YAML (Ansible/configuración). Las preguntas te muestran un fragmento JSON y piden identificar valores o estructuras.
