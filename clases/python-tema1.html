Python Esencial para Redes

Domina los fundamentos de Python orientados 100% a networking. Variables, strings, listas, diccionarios, loops y funciones — todos con ejemplos reales de IPs, configuraciones Cisco y automatización de red.

🐍 Python 3.x 🎯 CCNA Domain 6.1 ⏱ ~75 min lectura 💻 Ejercicios interactivos 🧪 Proyecto incluido

¿Por qué Python para Redes?

Los ingenieros de redes que no saben programar están siendo reemplazados por los que sí saben. Python es el lenguaje elegido por Cisco para automatización. Su sintaxis es clara, tiene librerías especializadas (netmiko, napalm, ncclient) y puedes aprender lo suficiente para automatizar tareas reales en una semana.

💡 Lo que aprenderás

Variables para guardar IPs, strings para comandos IOS, listas para inventarios, diccionarios para configuraciones, loops para iterar routers, y funciones para tareas reutilizables. Todo con código de red real.

Loops — Iterar sobre infraestructura

loops_red.py
# For loop básico
routers = ["192.168.1.1", "192.168.1.2", "192.168.1.3"]
for ip in routers:
    print(f"Conectando a {ip}...")

# enumerate() — índice y valor
for num, ip in enumerate(routers, start=1):
    print(f"[{num}/{len(routers)}] {ip}")

# range()
for vlan in range(10, 51, 10):
    print(f"vlan {vlan}")

# zip() — combinar listas
hostnames = ["R1", "R2", "R3"]
ips       = ["10.0.0.1", "10.0.0.2", "10.0.0.3"]
for host, ip in zip(hostnames, ips):
    print(f"{host} → {ip}")

🚀 Proyecto Final — Generador de Inventario

Script completo que integra variables, strings, listas, diccionarios, loops y funciones para generar un inventario de red automatizado.

inventario_red.py
#!/usr/bin/env python3

# Inventario de dispositivos
INVENTARIO = [
    {"hostname": "R1-CORE", "ip": "10.0.0.1", "tipo": "router"},
    {"hostname": "SW1-ACC", "ip": "10.0.0.10", "tipo": "switch"},
]

def validar_ip(ip):
    try:
        partes = ip.split(".")
        return len(partes) == 4 and all(0 <= int(p) <= 255 for p in partes)
    except:
        return False

def main():
    for disp in INVENTARIO:
        estado = "✅ OK" if validar_ip(disp["ip"]) else "❌ Error"
        print(f"{disp['hostname']:<12} {disp['ip']:<14} {estado}")

if __name__ == "__main__":
    main()

Guarda el código en inventario_red.py y ejecuta: python inventario_red.py
