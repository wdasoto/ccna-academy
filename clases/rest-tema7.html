PROYECTO — Dashboard de Red vía APIs

Consolida datos de múltiples controladores y dispositivos en un único panel ejecutivo. Consulta DNA Center para inventario y salud, RESTCONF para configuraciones IOS-XE en tiempo real, y genera informes HTML estáticos o un servidor Flask dinámico. El proyecto final que integra todo el M8.

📊 Dashboard 🎯 CCNA Domain 6.5 ⏱ ~120 min lectura 🐍 Python + Flask 🌐 Multi-API

Visión general del sistema

Este proyecto final del M8 integra dos mundos: la orquestación centralizada (DNA Center como fuente única de verdad para topología, inventario y salud) y la gestión directa de dispositivos (RESTCONF sobre IOS-XE para configuraciones y estadísticas de interfaces). El resultado es un dashboard unificado que los ingenieros de NOC pueden consultar cada mañana o mantener abierto en una pantalla de operaciones.

💡 Filosofía del diseño

Cada adapter es independiente y testeable. Si DNA Center no responde, el dashboard aún muestra datos de RESTCONF. Si un router está offline, se marca como unreachable pero no rompe el reporte. Este patrón se llama "graceful degradation" y es fundamental en sistemas de red.

Módulo DNA Center: inventario y salud

DNA Center es el cerebro del campus. A través de su API REST proporciona descubrimiento automatizado, topología, salud de dispositivos (Health Score), problemas activos (Issues) y configuraciones. Nuestro adapter encapsula toda la lógica de autenticación y normalización.

POST/dna/system/api/v1/auth/token
Obtiene el X-Auth-Token usando Basic Auth. Expira tras 1 hora de inactividad.
GET/dna/intent/api/v1/network-device
Lista todos los dispositivos gestionados: hostname, IP, plataforma, software, uptime, reachability.
GET/dna/intent/api/v1/interface
Todas las interfaces descubiertas con status administrativo/operacional, VLAN, duplex, velocidad.
GET/dna/intent/api/v1/issue
Issues activas priorizadas por severity: Critical, Major, Minor, Warning.

Implementación del adapter

adapters/dnac_client.py — Cliente DNA Center completo
#!/usr/bin/env python3
import requests
import os
from datetime import datetime


class DNACenterClient:
    """Adapter para Cisco DNA Center API v1."""

    def __init__(self, base_url, username, password, verify_ssl=True):
        self.base_url = base_url.rstrip("/")
        self.username = username
        self.password = password
        self.verify_ssl = verify_ssl
        self.session = requests.Session()
        self.session.verify = verify_ssl
        self.token = None
        self.token_expiry = None

        if not verify_ssl:
            requests.packages.urllib3.disable_warnings()

        self._authenticate()

    def _authenticate(self):
        """Obtiene token y configura header X-Auth-Token."""
        auth_url = f"{self.base_url}/dna/system/api/v1/auth/token"
        resp = self.session.post(
            auth_url,
            auth=(self.username, self.password),
            timeout=10
        )
        resp.raise_for_status()
        data = resp.json()
        self.token = data["Token"]
        self.session.headers.update({
            "X-Auth-Token": self.token,
            "Content-Type": "application/json",
            "Accept": "application/json"
        })
        self.token_expiry = datetime.now()
        print(f"🔐 DNA Center autenticado. Token vigente.")

    def _request(self, method, endpoint, **kwargs):
        """Wrapper con manejo de errores y reintentos simples."""
        url = f"{self.base_url}{endpoint}"
        try:
            r = self.session.request(method, url, timeout=30, **kwargs)
            r.raise_for_status()
            return r.json()
        except requests.exceptions.HTTPError as e:
            if r.status_code == 401:
                print("🔄 Token expirado. Reautenticando...")
                self._authenticate()
                return self._request(method, endpoint, **kwargs)
            raise

    # ═══════════════════════════════════════════════════
    # MÉTODOS PÚBLICOS
    # ═══════════════════════════════════════════════════

    def get_devices(self):
        """Retorna lista normalizada de dispositivos."""
        data = self._request("GET", "/dna/intent/api/v1/network-device")
        devices = []
        for d in data.get("response", []):
            devices.append({
                "source": "dnac",
                "id": d.get("id"),
                "hostname": d.get("hostname", "N/A"),
                "ip": d.get("managementIpAddress"),
                "platform": d.get("platformId", "Unknown"),
                "software": d.get("softwareVersion", "N/A"),
                "role": d.get("role", "Unknown"),
                "uptime": d.get("upTime", "N/A"),
                "reachability": d.get("reachabilityStatus", "Unknown"),
                "health": d.get("overallHealth", -1),
                "serial": d.get("serialNumber", "N/A")
            })
        return devices

    def get_interfaces(self, device_id=None):
        """Retorna interfaces con status operacional."""
        params = {"deviceId": device_id} if device_id else {}
        data = self._request("GET", "/dna/intent/api/v1/interface", params=params)
        interfaces = []
        for iface in data.get("response", []):
            interfaces.append({
                "source": "dnac",
                "device_id": iface.get("deviceId"),
                "name": iface.get("portName", "N/A"),
                "admin_status": iface.get("adminStatus", "unknown"),
                "oper_status": iface.get("status", "unknown"),
                "vlan": iface.get("vlanId", "N/A"),
                "speed": iface.get("speed", "N/A"),
                "duplex": iface.get("duplex", "N/A"),
                "mac": iface.get("macAddress", "N/A")
            })
        return interfaces

    def get_issues(self, severity=None):
        """Retorna issues activas, opcionalmente filtradas por severidad."""
        params = {"severity": severity} if severity else {}
        data = self._request("GET", "/dna/intent/api/v1/issue", params=params)
        issues = []
        for issue in data.get("response", []):
            issues.append({
                "source": "dnac",
                "name": issue.get("name", "Unknown"),
                "severity": issue.get("severity", "Unknown"),
                "category": issue.get("category", "General"),
                "device": issue.get("deviceId", "N/A"),
                "description": issue.get("description", ""),
                "timestamp": issue.get("issueOccurenceTime", "N/A")
            })
        return issues
✅ Buenas prácticas implementadas
  • Reautenticación automática: Si el token expira (HTTP 401), el cliente se reautentica sin interrumpir el flujo.
  • Normalización de datos: Todos los métodos retornan diccionarios planos con claves consistentes, independientemente del schema crudo de DNA Center.
  • Session reuse: Mantiene keep-alive y headers comunes entre peticiones.

RESTCONF sobre IOS-XE: modelo YANG

Mientras DNA Center ve la red desde arriba (orquestación), RESTCONF permite hablar directamente con el dispositivo. Usa el modelo de datos YANG (RFC 6020) para estructurar la configuración y el estado del equipo. Cada router/switch con IOS-XE 16.6+ expone un servidor RESTCONF en /restconf.

📡

ietf-interfaces

Modelo estándar IETF para interfaces: nombre, descripción, tipo, enabled, oper-status, estadísticas (in/out octets, errors, drops).

⚙️

Cisco-IOS-XE-native

Modelo propietario Cisco que expone toda la configuración nativa: hostname, VLANs, routing, ACLs, NAT, SNMP, usuarios.

📊

Cisco-IOS-XE-process-cpu

Estadísticas de CPU: uso por proceso, load average, historial de 5 segundos, 1 minuto y 5 minutos.

💾

Cisco-IOS-XE-memory

Utilización de memoria: total, used, free, hold, reservada por procesos.

Endpoints RESTCONF fundamentales

GET/restconf/data/ietf-interfaces:interfaces
Lista todas las interfaces con configuración y estado operacional.
GET/restconf/data/ietf-interfaces:interfaces/interface={name}
Detalle de una interfaz específica (ej. GigabitEthernet1).
GET/restconf/data/Cisco-IOS-XE-native:native/hostname
Hostname configurado en el dispositivo.
GET/restconf/data/Cisco-IOS-XE-process-cpu:cpu-usage
Uso actual de CPU con breakdown por proceso.

Implementación del adapter RESTCONF

adapters/restconf_client.py — Cliente IOS-XE RESTCONF
#!/usr/bin/env python3
import requests
import urllib3

class RestconfClient:
    """Adapter para dispositivos IOS-XE vía RESTCONF."""

    def __init__(self, host, username, password, port=443, verify_ssl=False):
        self.base_url = f"https://{host}:{port}/restconf/data"
        self.session = requests.Session()
        self.session.auth = (username, password)
        self.session.headers.update({
            "Accept": "application/yang-data+json",
            "Content-Type": "application/yang-data+json"
        })
        self.session.verify = verify_ssl

        if not verify_ssl:
            urllib3.disable_warnings(urllib3.exceptions.InsecureRequestWarning)

    def _get(self, yang_path):
        """GET genérico a un YANG model path."""
        url = f"{self.base_url}/{yang_path}"
        r = self.session.get(url, timeout=15)
        r.raise_for_status()
        return r.json()

    def get_hostname(self):
        data = self._get("Cisco-IOS-XE-native:native/hostname")
        return data.get("Cisco-IOS-XE-native:hostname", "N/A")

    def get_interfaces(self):
        """Retorna interfaces con estado y estadísticas."""
        data = self._get("ietf-interfaces:interfaces")
        interfaces = []
        for iface in data.get("ietf-interfaces:interface", []):
            stats = iface.get("statistics", {})
            interfaces.append({
                "source": "restconf",
                "name": iface.get("name", "N/A"),
                "description": iface.get("description", ""),
                "type": iface.get("type", "N/A"),
                "enabled": iface.get("enabled", False),
                "oper_status": iface.get("oper-status", "unknown"),
                "ip": self._extract_ip(iface),
                "in_octets": stats.get("in-octets", 0),
                "out_octets": stats.get("out-octets", 0),
                "in_errors": stats.get("in-errors", 0),
                "out_errors": stats.get("out-errors", 0)
            })
        return interfaces

    def _extract_ip(self, iface):
        """Extrae IP de la interfaz si tiene configuración IPv4."""
        ip_data = iface.get("ietf-ip:ipv4", {})
        addresses = ip_data.get("address", [])
        if addresses:
            return f"{addresses[0]['ip']}/{addresses[0].get('prefix-length','')}"
        return "N/A"

    def get_cpu_usage(self):
        """Retorna uso de CPU del dispositivo."""
        try:
            data = self._get("Cisco-IOS-XE-process-cpu:cpu-usage")
            cpu = data.get("Cisco-IOS-XE-process-cpu:cpu-usage", {})
            return {
                "five_seconds": cpu.get("cpu-utilization", {}).get("five-seconds", "N/A"),
                "one_minute": cpu.get("cpu-utilization", {}).get("one-minute", "N/A"),
                "five_minutes": cpu.get("cpu-utilization", {}).get("five-minutes", "N/A")
            }
        except Exception:
            return {"five_seconds": "unavailable", "one_minute": "unavailable"}

    def get_memory(self):
        """Retorna estadísticas de memoria."""
        try:
            data = self._get("Cisco-IOS-XE-memory:memory-statistics")
            mem = data.get("Cisco-IOS-XE-memory:memory-statistics", {})
            return {
                "total": mem.get("total-memory", 0),
                "used": mem.get("used-memory", 0),
                "free": mem.get("free-memory", 0)
            }
        except Exception:
            return {"total": 0, "used": 0, "free": 0}
⚡ Header Accept crítico en RESTCONF

RESTCONF requiere explícitamente Accept: application/yang-data+json. Si omites este header o usas application/json genérico, IOS-XE puede retornar XML o un error 406 Not Acceptable. El header Content-Type también debe ser application/yang-data+json en POST/PUT/PATCH.

Data Merger: uniendo mundos

El desafío central de un dashboard multi-API es que cada fuente habla un idioma diferente. DNA Center usa managementIpAddress y platformId; RESTCONF usa name y type. El Data Merger normaliza, deduplica y enriquece la información para presentar una vista única de la red.

Reglas de consolidación

  • Clave de matching: Usamos la dirección IP de gestión o el hostname como clave primaria para unificar registros.
  • Prioridad de fuentes: Si un dispositivo existe en DNA Center y RESTCONF, DNA Center tiene prioridad para metadatos (plataforma, software), pero RESTCONF enriquece con datos en tiempo real (CPU, memoria, stats de interfaces).
  • Marcado de fuente: Cada registro conserva el campo sources: ["dnac", "restconf"] para trazabilidad.
  • Cálculo de métricas agregadas: Porcentaje de dispositivos alcanzables, interfaces operativas, issues críticas, uso promedio de CPU.

Implementación del Data Merger

core/merger.py — Motor de consolidación
#!/usr/bin/env python3
from collections import defaultdict


class DataMerger:
    """Consolida datos de DNA Center y RESTCONF en una vista unificada."""

    def __init__(self):
        self.devices = {}
        self.interfaces = []
        self.issues = []
        self.metrics = {}

    def ingest_dnac(self, devices, interfaces, issues):
        """Ingesta datos desde DNA Center."""
        for d in devices:
            key = d["ip"] or d["hostname"]
            self.devices[key] = {
                "hostname": d["hostname"],
                "ip": d["ip"],
                "platform": d["platform"],
                "software": d["software"],
                "role": d["role"],
                "reachability": d["reachability"],
                "health": d["health"],
                "serial": d["serial"],
                "sources": ["dnac"],
                "cpu": None,
                "memory": None
            }
        self.interfaces.extend(interfaces)
        self.issues.extend(issues)

    def ingest_restconf(self, host_ip, hostname, interfaces, cpu, memory):
        """Ingesta datos desde un dispositivo IOS-XE vía RESTCONF."""
        key = host_ip
        if key in self.devices:
            # Enriquecer registro existente de DNAC
            self.devices[key]["cpu"] = cpu
            self.devices[key]["memory"] = memory
            if "restconf" not in self.devices[key]["sources"]:
                self.devices[key]["sources"].append("restconf")
        else:
            # Dispositivo solo en RESTCONF (no gestionado por DNAC)
            self.devices[key] = {
                "hostname": hostname,
                "ip": host_ip,
                "platform": "IOS-XE",
                "software": "Unknown",
                "role": "Edge",
                "reachability": "Reachable",
                "health": -1,
                "serial": "N/A",
                "sources": ["restconf"],
                "cpu": cpu,
                "memory": memory
            }

        # Añadir interfaces con prefijo de fuente
        for iface in interfaces:
            iface["device_ip"] = host_ip
            iface["device_hostname"] = hostname
            self.interfaces.append(iface)

    def compute_metrics(self):
        """Calcula métricas agregadas del estado de la red."""
        total = len(self.devices)
        reachable = sum(1 for d in self.devices.values()
                          if d["reachability"] == "Reachable")
        healthy = sum(1 for d in self.devices.values()
                        if d["health"] >= 80 or d["health"] == -1)

        up_ifaces = sum(1 for i in self.interfaces
                          if i.get("oper_status") in ["up", "ok"])

        critical_issues = sum(1 for issue in self.issues
                                if issue.get("severity") == "CRITICAL")

        self.metrics = {
            "total_devices": total,
            "reachable_pct": round((reachable / total * 100), 1) if total else 0,
            "healthy_pct": round((healthy / total * 100), 1) if total else 0,
            "total_interfaces": len(self.interfaces),
            "up_interfaces": up_ifaces,
            "up_interfaces_pct": round((up_ifaces / len(self.interfaces) * 100), 1) if self.interfaces else 0,
            "total_issues": len(self.issues),
            "critical_issues": critical_issues
        }

    def get_devices_list(self):
        return list(self.devices.values())

    def get_interfaces_list(self):
        return self.interfaces

    def get_issues_list(self):
        return self.issues

    def get_metrics(self):
        return self.metrics

Motor de alertas unificado

Con los datos consolidados, el motor de alertas evalúa reglas de negocio y genera notificaciones clasificadas por severidad. Estas alertas aparecerán en el dashboard con color distintivo.

core/alert_engine.py — Generador de alertas
#!/usr/bin/env python3


class AlertEngine:
    """Evalúa reglas sobre datos consolidados y genera alertas."""

    RULES = [
        {"name": "Device Unreachable", "severity": "CRITICAL",
         "check": lambda d: d.get("reachability") != "Reachable"},
        {"name": "Low Health Score", "severity": "WARNING",
         "check": lambda d: 0 <= d.get("health", 100) < 70},
        {"name": "High CPU Usage", "severity": "WARNING",
         "check": lambda d: d.get("cpu") and d["cpu"].get("five_seconds", 0) > 80},
        {"name": "Interface Down", "severity": "MINOR",
         "check": lambda i: i.get("oper_status") in ["down", "lower-layer-down"]},
    ]

    def evaluate(self, devices, interfaces):
        """Evalúa todas las reglas y retorna alertas."""
        alerts = []

        for device in devices:
            for rule in self.RULES[:3]:
                if rule["check"](device):
                    alerts.append({
                        "severity": rule["severity"],
                        "category": "Device",
                        "target": device.get("hostname", device.get("ip")),
                        "message": rule["name"],
                        "source": ",".join(device.get("sources", []))
                    })

        for iface in interfaces:
            rule = self.RULES[3]
            if rule["check"](iface):
                alerts.append({
                    "severity": rule["severity"],
                    "category": "Interface",
                    "target": f"{iface.get('device_hostname','?')}:{iface['name']}",
                    "message": rule["name"],
                    "source": "restconf"
                })

        # Ordenar: CRITICAL > WARNING > MINOR
        severity_order = {"CRITICAL": 0, "WARNING": 1, "MINOR": 2}
        alerts.sort(key=lambda x: severity_order.get(x["severity"], 99))
        return alerts

  • Auditorías de compliance donde necesitas un archivo inmutable con timestamp.

#!/usr/bin/env python3
import os
import json
from datetime import datetime
from jinja2 import Environment, FileSystemLoader

from adapters.dnac_client import DNACenterClient
from adapters.restconf_client import RestconfClient
from core.merger import DataMerger
from core.alert_engine import AlertEngine

def main():
    print("🚀 Iniciando generación de reporte de red...")

    # ── CONFIGURACIÓN ──
    merger = DataMerger()

    # 1. DNA Center
    dnac = DNACenterClient(
        base_url=os.getenv("DNAC_URL", "https://sandboxdnac.cisco.com"),
        username=os.getenv("DNAC_USER", "devnetuser"),
        password=os.getenv("DNAC_PASS", "Cisco123!"),
        verify_ssl=False
    )
    dnac_devices = dnac.get_devices()
    dnac_interfaces = dnac.get_interfaces()
    dnac_issues = dnac.get_issues()
    merger.ingest_dnac(dnac_devices, dnac_interfaces, dnac_issues)
    print(f"✅ DNA Center: {len(dnac_devices)} dispositivos")

    # 2. RESTCONF (lista de routers edge)
    edge_routers = json.loads(os.getenv("EDGE_ROUTERS", "[]"))
    for router in edge_routers:
        try:
            rc = RestconfClient(
                host=router["ip"],
                username=router["user"],
                password=router["pass"],
                verify_ssl=False
            )
            merger.ingest_restconf(
                host_ip=router["ip"],
                hostname=rc.get_hostname(),
                interfaces=rc.get_interfaces(),
                cpu=rc.get_cpu_usage(),
                memory=rc.get_memory()
            )
            print(f"✅ RESTCONF {router['ip']}: OK")
        except Exception as e:
            print(f"⚠️  RESTCONF {router['ip']}: {e}")

    # 3. Consolidar
    merger.compute_metrics()
    devices = merger.get_devices_list()
    interfaces = merger.get_interfaces_list()
    metrics = merger.get_metrics()

    # 4. Evaluar alertas
    engine = AlertEngine()
    alerts = engine.evaluate(devices, interfaces)

    # 5. Renderizar HTML
    env = Environment(loader=FileSystemLoader("templates"))
    template = env.get_template("report_static.html")
    html_out = template.render(
        timestamp=datetime.now().strftime("%Y-%m-%d %H:%M:%S"),
        devices=devices,
        interfaces=interfaces,
        metrics=metrics,
        alerts=alerts
    )

    filename = f"reports/network_dashboard_{datetime.now():%Y%m%d_%H%M}.html"
    os.makedirs("reports", exist_ok=True)
    with open(filename, "w", encoding="utf-8") as f:
        f.write(html_out)

    print(f"🎉 Reporte generado: {filename}")
    print(f"   Dispositivos: {metrics['total_devices']} | Alertas: {len(alerts)}")

if __name__ == "__main__":
    main()

import os

import time

from flask import Flask, render_template

# ── Estado global en memoria (cache) ──
dashboard_data = {
    "devices": [],
    "interfaces": [],
    "metrics": {},
    "alerts": [],
    "last_update": "Nunca"
}


def refresh_data():
    """Función que recarga datos de todas las fuentes."""
    global dashboard_data
    print(f"[{datetime.now()}] 🔄 Refrescando datos...")

    merger = DataMerger()

    # DNA Center
    try:
        dnac = DNACenterClient(
            os.getenv("DNAC_URL", "https://sandboxdnac.cisco.com"),

            dnac.get_devices(),
            dnac.get_interfaces(),
            dnac.get_issues()
        )
    except Exception as e:
        print(f"⚠️ DNA Center error: {e}")

    # RESTCONF routers
    for r in json.loads(os.getenv("EDGE_ROUTERS", "[]")):
        try:
            rc = RestconfClient(r["ip"], r["user"], r["pass"], verify_ssl=False)
            merger.ingest_restconf(
                r["ip"], rc.get_hostname(),
                rc.get_interfaces(), rc.get_cpu_usage(), rc.get_memory()
            )
        except Exception as e:

    merger.compute_metrics()
    devices = merger.get_devices_list()
    interfaces = merger.get_interfaces_list()
    alerts = AlertEngine().evaluate(devices, interfaces)

    dashboard_data = {
        "devices": devices,
        "interfaces": interfaces,
        "metrics": merger.get_metrics(),
        "alerts": alerts,
        "last_update": datetime.now().strftime("%Y-%m-%d %H:%M:%S")
    }
    print(f"✅ Datos actualizados. {len(devices)} devs, {len(alerts)} alertas.")


def background_refresh(interval=300):
    """Hilo que refresca datos cada N segundos."""
    while True:
        refresh_data()
        time.sleep(interval)


# ── RUTAS FLASK ──
@app.route("/")
def dashboard():
    return render_template(
        "dashboard_live.html",
        **dashboard_data
    )


@app.route("/api/refresh", methods=["POST"])
def api_refresh():
    refresh_data()
    return {"status": "ok", "last_update": dashboard_data["last_update"]}


if __name__ == "__main__":
    # Carga inicial antes de arrancar
    refresh_data()

    # Iniciar hilo de refresco automático (5 min)
    t = threading.Thread(target=background_refresh, daemon=True)
    t.start()

    app.run(host="0.0.0.0", port=5000, debug=False)

Sistema de alertas unificado

Un dashboard sin alertas es solo una tabla bonita. El verdadero valor está en detectar anomalías automáticamente y presentarlas priorizadas. Nuestro sistema combina issues de DNA Center con reglas propias sobre datos RESTCONF (CPU, memoria, interfaces down).

Tipos de alertas y severidad

🔴

CRITICAL

Dispositivo no alcanzable, interfaz core down, fallo de hardware detectado. Requiere acción inmediata.

🟡

WARNING

Health score bajo, CPU > 80%, memoria saturada, errores de CRC elevados. Monitorear de cerca.

🔵

MINOR

Interfaz de acceso down, versión de software desactualizada, configuración no óptima.

import requests

def send_slack_alert(webhook_url, alert):
    color = {
        "CRITICAL": "#f87171",

    }

dashboard.py — Orquestador unificado
#!/usr/bin/env python3
"""
dashboard.py
CCNA Academy M8 T7 — Dashboard de Red Multi-API
────────────────────────────────────────────────
Orquesta DNA Center + RESTCONF para generar reportes HTML
estáticos o un servidor Flask en tiempo real.

Uso:
  python dashboard.py --mode static    # Genera reporte batch
  python dashboard.py --mode flask     # Levanta servidor en :5000
"""

import argparse
import os
import json
import time
import threading
from datetime import datetime

from jinja2 import Environment, FileSystemLoader
from flask import Flask, render_template as flask_render

from adapters.dnac_client import DNACenterClient
from adapters.restconf_client import RestconfClient
from core.merger import DataMerger
from core.alert_engine import AlertEngine


def collect_data():
    """Recolecta datos de todas las fuentes configuradas."""
    merger = DataMerger()

    # ── DNA Center ──
    try:
        dnac = DNACenterClient(
            os.getenv("DNAC_URL", "https://sandboxdnac.cisco.com"),
            os.getenv("DNAC_USER", "devnetuser"),
            os.getenv("DNAC_PASS", "Cisco123!"),
            verify_ssl=False
        )
        merger.ingest_dnac(
            dnac.get_devices(),
            dnac.get_interfaces(),
            dnac.get_issues()
        )
    except Exception as e:
        print(f"[DNA-C] Error: {e}")

    # ── RESTCONF Edge Routers ──
    routers = json.loads(os.getenv("EDGE_ROUTERS", "[]"))
    for r in routers:
        try:
            rc = RestconfClient(r["ip"], r["user"], r["pass"], verify_ssl=False)
            merger.ingest_restconf(
                r["ip"], rc.get_hostname(),
                rc.get_interfaces(), rc.get_cpu_usage(), rc.get_memory()
            )
        except Exception as e:
            print(f"[RESTCONF {r['ip']}] Error: {e}")

    merger.compute_metrics()
    devices = merger.get_devices_list()
    interfaces = merger.get_interfaces_list()
    alerts = AlertEngine().evaluate(devices, interfaces)

    return {
        "devices": devices,
        "interfaces": interfaces,
        "metrics": merger.get_metrics(),
        "alerts": alerts,
        "timestamp": datetime.now().strftime("%Y-%m-%d %H:%M:%S")
    }


def run_static():
    """Genera reporte HTML estático en disco."""
    print("📄 Modo: REPORTE ESTÁTICO")
    data = collect_data()

    env = Environment(loader=FileSystemLoader("templates"))
    template = env.get_template("report_static.html")
    html = template.render(**data)

    os.makedirs("reports", exist_ok=True)
    fname = f"reports/dashboard_{datetime.now():%Y%m%d_%H%M%S}.html"
    with open(fname, "w", encoding="utf-8") as f:
        f.write(html)

    print(f"✅ Reporte guardado: {fname}")
    print(f"   Dispositivos: {data['metrics']['total_devices']}")
    print(f"   Alertas: {len(data['alerts'])}")


def run_flask():
    """Levanta servidor Flask con refresh automático."""
    print("🌐 Modo: FLASK LIVE DASHBOARD")
    app = Flask(__name__)
    cache = {"data": collect_data()}

    def refresher():
        while True:
            time.sleep(300)
            try:
                cache["data"] = collect_data()
                print(f"[{datetime.now()}] 🔄 Cache actualizada")
            except Exception as e:
                print(f"Refresh error: {e}")

    t = threading.Thread(target=refresher, daemon=True)
    t.start()

    @app.route("/")
    def index():
        return flask_render("dashboard_live.html", **cache["data"])

    @app.route("/api/status")
    def api_status():
        return {
            "devices": cache["data"]["metrics"]["total_devices"],
            "alerts": len(cache["data"]["alerts"]),
            "last_update": cache["data"]["timestamp"]
        }

    app.run(host="0.0.0.0", port=5000)


if __name__ == "__main__":
    parser = argparse.ArgumentParser(description="Dashboard de Red CCNA")
    parser.add_argument(
        "--mode",
        choices=["static", "flask"],
        default="static",
        help="Modo de ejecución"
    )
    args = parser.parse_args()

    if args.mode == "static":
        run_static()
    else:
        run_flask()
