Gestor de Red Completo

Proyecto Final integrador: Conecta a N dispositivos simultáneamente, automatiza backups, verifica configuraciones con NAPALM, parsea outputs complejos y genera reportes HTML profesionales. Todo en un único sistema de gestión enterprise.

🐍 Python 3.x 🔗 Netmiko + NAPALM 📄 Jinja2 Templates 🌐 Reporte HTML ⚡ Multi-hilo ⏱ ~90 min proyecto

Arquitectura del Gestor Enterprise

Este proyecto integra todo lo aprendido en M8 en una sola aplicación de producción. No es un script simple: es un sistema modular con separación de responsabilidades, manejo de errores robusto, logging profesional y generación de reportes HTML estilo dashboard.

Componentes del Sistema

📋
INVENTARIO
YAML centralizado con credenciales cifradas, grupos de dispositivos y variables personalizadas por plataforma.
🔌

🧪

Verificación de estado, getters de información estructurada y validación de configuración.
💾
BACKUP
Snapshots automáticos con timestamp, compresión gzip y almacenamiento en estructura de directorios fecha/hora.
🔍
TEXTFSM
Parseo de outputs de show commands a diccionarios estructurados para análisis programático.
📊
JINJA2 + HTML
Templates profesionales que generan reportes HTML responsivos con tablas, estado de salud y gráficos CSS.
💡 Flujo de Ejecución
  1. Carga: Lee inventory.yml y valida sintaxis
  2. Discovery: Prueba conectividad ICMP a todos los hosts
  3. Conexión: Establece SSH paralelo a dispositivos alcanzables
  4. Recolección: Ejecuta getters NAPALM + comandos personalizados
  5. Parseo: Convierte outputs crudos a estructuras de datos
  6. Análisis: Detecta drift de configuración, errores de spanning-tree, rutas faltantes
  7. Backup: Guarda running-config de cada dispositivo
  8. Reporte: Genera reporte_ejecutivo_20240115_143022.html

Cargando el Inventario en Python

lib/inventory_loader.py
import yaml
from pathlib import Path
from jinja2 import Template

class InventoryManager:
    """Carga y enriquece inventario YAML"""
    
    def __init__(self, inv_path="inventory/production.yml", 
                 sec_path="inventory/secrets.yml"):
        with open(inv_path) as f:
            self.raw = yaml.safe_load(f)
        with open(sec_path) as f:
            self.secrets = yaml.safe_load(f)
    
    def get_all_devices(self):
        """Retorna lista plana de dispositivos con sus credenciales"""
        devices = []
        for group_name, group_data in self.raw['all']['children'].items():
            group_vars = group_data.get('vars', {})
            for hostname, host_vars in group_data.get('hosts', {}).items():
                dev = {
                    'hostname': hostname,
                    'group': group_name,
                    **group_vars,  # Herencia de grupo
                    **host_vars    # Variables específicas
                }
                # Merge credentials
                dev.update(self.secrets['credentials'].get(group_name, 
                              self.secrets['credentials']['default']))
                devices.append(dev)
        return devices
    
    def filter_by_role(self, role):
        """Filtra dispositivos por rol"""
        return [d for d in self.get_all_devices() if d.get('role') == role]

Conectividad SSH Multi-Hilo con Netmiko

Conexiones paralelas usando concurrent.futures.ThreadPoolExecutor. Esto permite gestionar 50+ dispositivos simultáneamente sin bloquear el hilo principal.

lib/connector.py — Clase Connector robusta
from netmiko import ConnectHandler, NetMikoTimeoutException, 
                NetMikoAuthenticationException
from concurrent.futures import ThreadPoolExecutor, as_completed
import logging

class NetworkConnector:
    """Wrapper de Netmiko con manejo de errores y logging"""
    
    def __init__(self, max_workers=10):
        self.max_workers = max_workers
        self.results = {}
        self.logger = logging.getLogger(__name__)
    
    def connect_single(self, device):
        """Conecta a un dispositivo y retorna el handler"""
        try:
            conn = ConnectHandler(
                device_type=device['device_type'],
                host=device['ip'],
                username=device['username'],
                password=device['password'],
                secret=device.get('enable_secret', ''),
                timeout=30,
                banner_timeout=15
            )
            # Entrar a enable mode si es necesario
            if conn.find_prompt().endswith('>'):
                conn.enable()
            return {'status': 'success', 'conn': conn, 'device': device}
        except NetMikoTimeoutException:
            return {'status': 'timeout', 'device': device, 'error': 'Connection timeout'}
        except NetMikoAuthenticationException:
            return {'status': 'auth_failed', 'device': device, 'error': 'Auth failed'}
        except Exception as e:
            return {'status': 'error', 'device': device, 'error': str(e)}
    
    def execute_command(self, device, command):
        """Ejecuta un comando en un dispositivo específico"""
        result = self.connect_single(device)
        if result['status'] == 'success':
            try:
                output = result['conn'].send_command(command, 
                    use_textfsm=True, textfsm_template="templates/textfsm/")
                result['conn'].disconnect()
                return {
                    'device': device['hostname'],
                    'command': command,
                    'output': output,
                    'status': 'success'
                }
            except Exception as e:
                return {'device': device['hostname'], 'status': 'error', 'error': str(e)}
        return result
    
    def mass_execute(self, devices, command, max_workers=None):
        """Ejecuta un comando en paralelo sobre N dispositivos"""
        workers = max_workers or self.max_workers
        results = []
        
        with ThreadPoolExecutor(max_workers=workers) as executor:
            future_to_dev = {
                executor.submit(self.execute_command, dev, command): dev 
                for dev in devices
            }
            for future in as_completed(future_to_dev):
                results.append(future.result())
        
        return results
⚡ Concurrencia vs Paralelismo

ThreadPoolExecutor usa hilos (threads), no procesos. Esto es ideal para I/O-bound tasks como SSH, donde la CPU espera datos de red. Para 50+ dispositivos, usa max_workers=20. Para más de 100, considera asyncio con asyncssh.

Verificación con NAPALM

NAPALM proporciona una API unificada para obtener información estructurada (getters) y validar configuración sin preocuparnos por el sistema operativo subyacente.

lib/compliance.py — Verificación de cumplimiento
from napalm import get_network_driver
from napalm.base.exceptions import ConnectionException

class ComplianceChecker:
    """Verifica políticas de seguridad y configuración"""
    
    def __init__(self, device):
        self.device = device
        self.driver = get_network_driver(device['device_type'].replace("cisco_ios", "ios"))
        self.results = {}
    
    def check_ssh_only(self):
        """Verifica que solo SSH esté habilitado (no Telnet)"""
        try:
            with self.driver(hostname=self.device['ip'],
                           username=self.device['username'],
                           password=self.device['password']) as device:
                config = device.get_config()['running']
                has_telnet = 'transport input telnet' in config.lower()
                return {
                    'policy': 'ssh_only',
                    'compliant': not has_telnet,
                    'details': 'Telnet found' if has_telnet else 'SSH only'
                }
        except Exception as e:
            return {'policy': 'ssh_only', 'compliant': False, 'error': str(e)}
    
    def check_firmware(self, min_version="16.9"):
        """Verifica versión mínima de IOS"""
        with self.driver(...) as device:
            facts = device.get_facts()
            current = facts['os_version'].split('.')[:2]
            required = min_version.split('.')
            return {
                'current': facts['os_version'],
                'compliant': tuple(current) >= tuple(required)
            }

Sistema de Backup Automatizado

Backups con timestamp, compresión y rotación automática. Guardamos metadatos (hash MD5) para detectar cambios no autorizados.

lib/backup_manager.py — Gestión de configuraciones
from datetime import datetime
from pathlib import Path
import gzip
import hashlib
import json

class BackupManager:
    """Backups con metadatos y diff"""
    
    def __init__(self, base_path="backups"):
        self.base = Path(base_path)
        self.base.mkdir(exist_ok=True)
    
    def backup_device(self, hostname, config_text):
        """Guarda configuración con metadatos"""
        timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
        date_folder = datetime.now().strftime("%Y-%m-%d")
        
        # Estructura: backups/2024-01-15/R1-CORE_20240115_143022.cfg.gz
        day_dir = self.base / date_folder
        day_dir.mkdir(exist_ok=True)
        
        filename = f"{hostname}_{timestamp}.cfg"
        filepath = day_dir / filename
        
        # Comprimir y guardar
        with gzip.open(str(filepath) + '.gz', 'wt') as f:
            f.write(config_text)
        
        # Calcular hash para integridad
        file_hash = hashlib.md5(config_text.encode()).hexdigest()
        
        # Metadatos JSON
        meta = {
            'hostname': hostname,
            'timestamp': timestamp,
            'hash': file_hash,
            'size': len(config_text),
            'path': str(filepath) + '.gz'
        }
        
        with open(day_dir / f"{hostname}_{timestamp}.json", 'w') as f:
            json.dump(meta, f, indent=2)
        
        return meta

lib/parser.py — Motor de parseo
import textfsm
from io import StringIO

class OutputParser:
    """Parsea outputs usando TextFSM"""
    
    def parse_routing_table(self, raw_output):
        """Extrae rutas a estructura JSON"""
        template = "templates/textfsm/cisco_ios_show_ip_route.textfsm"
        
        with open(template) as f:
            re_table = textfsm.TextFSM(f)
            data = re_table.ParseText(raw_output)
            
            # Convertir a lista de diccionarios
            keys = ['protocol', 'network', 'mask', 'distance', 
                   'metric', 'nexthop', 'interface']
            return [dict(zip(keys, row)) for row in data]
    
    def analyze_route_health(self, routes):
        """Detecta problemas en la tabla de routing"""
        issues = []
        for route in routes:
            if route['protocol'] == 'S' and route['distance'] != '1':
                issues.append(f"Static route {route['network']} AD != 1")
            if route['protocol'] == 'B' and int(route['metric']) > 1000:
                issues.append(f"BGP route {route['network']} high metric")
        return issues

from jinja2 import Environment, FileSystemLoader
from datetime import datetime
import json

class HTMLReporter:
    """Genera reportes ejecutivos en HTML"""
    
    def __init__(self, template_dir="templates"):
        self.env = Environment(loader=FileSystemLoader(template_dir))
    
    def generate(self, data, output_file="reports/network_report.html"):
        """Genera el reporte final"""
        template = self.env.get_template('report_template.html')
        
        context = {
            'generated_at': datetime.now().strftime("%Y-%m-%d %H:%M:%S"),
            'summary': data['summary'],
            'devices': data['devices'],
            'compliance': data['compliance'],
            'routes': data.get('routes', []),
            'alerts', [])
        }
        
        html_output = template.render(**context)
        
        Path(output_file).parent.mkdir(parents=True, exist_ok=True)
        with open(output_file, 'w', encoding='utf-8') as f:
            f.write(html_output)
        
        return output_file

🚀 Código Completo del Gestor de Red

Este es el script principal que integra todos los módulos. Ejecuta el flujo completo: carga inventario, conecta a dispositivos, recolecta datos, verifica cumplimiento, hace backups y genera el reporte HTML.

gestor_red.py — Script Principal Integrado (250 líneas)
#!/usr/bin/env python3
"""
Network Manager Enterprise v1.0
─────────────────────────────────────────────────
Sistema integrado de gestión de red que combina:
  - Inventario YAML dinámico
  - Conexiones SSH paralelas (Netmiko)
  - Verificación NAPALM
  - Backups automatizados con compresión
  - Parseo TextFSM de outputs
  - Reportes HTML profesionales
─────────────────────────────────────────────────
"""

import yaml
import logging
import sys
from datetime import datetime
from pathlib import Path
from concurrent.futures import ThreadPoolExecutor, as_completed

# Importar nuestros módulos (asumiendo estructura de paquete)
try:
    from lib.inventory_manager import InventoryManager
    from lib.connector import NetworkConnector
    from lib.backup_manager import BackupManager
    from lib.compliance import ComplianceChecker
    from lib.reporter import HTMLReporter
    from lib.parser import OutputParser
except ImportError:
    print("Error: Ejecuta desde el directorio raíz del proyecto")
    sys.exit(1)

# Configuración de logging
logging.basicConfig(
    level=logging.INFO,
    format='%(asctime)s - %(name)s - %(levelname)s - %(message)s',
    handlers=[
        logging.FileHandler('logs/gestor_red.log'),
        logging.StreamHandler()
    ]
)
logger = logging.getLogger('NetworkManager')

class NetworkManager:
    """Clase principal orquestadora"""
    
    def __init__(self):
        self.inv = InventoryManager()
        self.connector = NetworkConnector(max_workers=10)
        self.backup = BackupManager()
        self.reporter = HTMLReporter()
        self.parser = OutputParser()
        self.results = {
            'devices': [],
            'compliance': [],
            'backups': [],
            'alerts': [],
            'summary': {}
        }
    
    def discovery(self):
        """Fase 1: Descubrimiento y conectividad"""
        logger.info("=== FASE 1: DISCOVERY ===")
        devices = self.inv.get_all_devices()
        logger.info(f"Cargados {len(devices)} dispositivos del inventario")
        return devices
    
    def collect_data(self, devices):
        """Fase 2: Recolección paralela de datos"""
        logger.info("=== FASE 2: DATA COLLECTION ===")
        
        def process_device(dev):
            """Procesa un dispositivo individual"""
            record = {
                'hostname': dev['hostname'],
                'ip': dev['ip'],
                'group': dev['group'],
                'reachable': False,
                'facts': {},
                'interfaces': [],
                'routes': []
            }
            
            try:
                # Conectar y obtener facts
                conn = self.connector.connect_single(dev)
                if conn['status'] == 'success':
                    record['reachable'] = True
                    device_conn = conn['conn']
                    
                    # Backup de configuración
                    config = device_conn.send_command("show running-config")
                    backup_meta = self.backup.backup_device(dev['hostname'], config)
                    record['backup_hash'] = backup_meta['hash']
                    
                    # Comandos específicos con parseo
                    route_output = device_conn.send_command("show ip route")
                    record['routes'] = self.parser.parse_routing_table(route_output)
                    
                    # Facts básicos                    
                    show_ver = device_conn.send_command("show version", use_textfsm=True)
                    if show_ver and len(show_ver) > 0:
                        record['facts'] = show_ver[0]
                    
                    device_conn.disconnect()
                    logger.info(f"✅ {dev['hostname']}: Datos recolectados")
                else:
                    record['error'] = conn.get('error', 'Connection failed')
                    logger.warning(f"❌ {dev['hostname']}: {record['error']}")
                    
            except Exception as e:
                record['error'] = str(e)
                logger.error(f"💥 {dev['hostname']}: Excepción - {e}")
            
            return record
        
        # Ejecutar en paralelo
        with ThreadPoolExecutor(max_workers=10) as executor:
            futures = {executor.submit(process_device, dev): dev for dev in devices}
            for future in as_completed(futures):
                result = future.result()
                self.results['devices'].append(result)
    
    def check_compliance(self):
        """Fase 3: Verificación de políticas"""
        logger.info("=== FASE 3: COMPLIANCE CHECK ===")
        
        for dev in self.results['devices']:
            if dev['reachable']:
                checker = ComplianceChecker(dev)
                ssh_check = checker.check_ssh_only()
                self.results['compliance'].append({
                    'hostname': dev['hostname'],
                    'checks': [ssh_check]
                })
                
                if not ssh_check['compliant']:
                    self.results['alerts'].append({
                        'severity': 'high',
                        'device': dev['hostname'],
                        'message': ssh_check['details']
                    })
    
    def generate_summary(self):
        """Genera métricas resumen"""
        total = len(self.results['devices'])
        reachable = sum(1 for d in self.results['devices'] if d['reachable'])
        
        self.results['summary'] = {
            'total_devices': total,
            'reachable': reachable,
            'unreachable': total - reachable,
            0 if total == 0 else 
                (sum(1 for c in self.results['compliance'] 
                 if all(x['compliant'] for x in c['checks'])) / total * 100)
        }
    
    def generate_report(self):
        """Fase 4: Generación de reporte"""
        logger.info("=== FASE 4: REPORT GENERATION ===")
        
        timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
        report_path = f"reports/network_report_{timestamp}.html"
        
        output = self.reporter.generate(self.results, report_path)
        logger.info(f"📊 Reporte generado: {output}")
        return output
    
    def run(self):
        """Flujo principal"""
        logger.info("🚀 Iniciando Network Manager Enterprise")
        
        # Crear directorios necesarios
        Path("logs").mkdir(exist_ok=True)
        Path("reports").mkdir(exist_ok=True)
        
        # Pipeline
        devices = self.discovery()
        self.collect_data(devices)
        self.check_compliance()
        self.generate_summary()
        report_file = self.generate_report()
        
        # Resumen en consola
        print("\n" + "="*50)
        print("EJECUCIÓN COMPLETADA")
        print("="*50)
        print(f"Total dispositivos: {self.results['summary']['total_devices']}")
        print(f"Dispositivos OK: {self.results['summary']['reachable']}")
        print(f"Alertas: {len(self.results['alerts'])}")
        print(f"Reporte: {report_file}")

    manager = NetworkManager()
