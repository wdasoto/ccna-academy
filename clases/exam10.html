FUNDAMENTOS
Pregunta 1
¿Cuántos hosts utilizables tiene una red /26?
✓ Respuesta: B
/26 = 2^6 - 2 = 62 hosts. Fórmula: 2^(32-prefijo) - 2.
SWITCHING
Pregunta 2
¿Qué protocolo previene loops de capa 2?
✓ Respuesta: B
STP (802.1D) previene loops bloqueando puertos redundantes. Variantes: RSTP (802.1w), MST (802.1s).
ROUTING
Pregunta 3
¿Qué distancia administrativa tiene OSPF?
✓ Respuesta: B
AD: Conectada=0, Estática=1, EIGRP=90, OSPF=110, RIP=120, Externa=170, Desconocida=255.
SERVICIOS IP
Pregunta 4
¿Qué hace el comando ip helper-address 10.0.0.1?
✓ Respuesta: A
ip helper-address = DHCP relay agent. Reenvía broadcast DHCP/TFTP/DNS como unicast.
SEGURIDAD
Pregunta 5
¿Qué acción de Port Security permite tráfico pero genera syslog?
✓ Respuesta: B
restrict descarta frames violatorios + log. protect=descarta sin log. shutdown=err-disable.
IPv6
Pregunta 6
¿Qué dirección IPv6 representa ::1?
✓ Respuesta: A
::1 = loopback IPv6 (equivale a 127.0.0.1 en IPv4).
WIRELESS
Pregunta 7
¿Cuántos canales no-solapados hay en 2.4 GHz?
✓ Respuesta: A
2.4 GHz: 3 canales (1, 6, 11). 5 GHz: 24 canales no-solapados.
AUTOMATIZACIÓN
Pregunta 8
¿Qué método HTTP REST actualiza un recurso existente?
✓ Respuesta: C
PUT reemplaza recurso completo. PATCH actualiza parcialmente.
CONFIGURACIÓN
Pregunta 50
¿Qué comando guarda la configuración running-config a startup-config?
✓ Respuesta: D
Todas son válidas. copy run start es recomendado (CLI moderna). wr y write memory son legacy.
