Pregunta 1
¿Cuál es la Distancia Administrativa (AD) predeterminada de una ruta estática en un router Cisco?
✓ RESPUESTA: B
Las rutas estáticas tienen una AD de 1. Las interfaces conectadas directamente tienen AD 0.
Pregunta 2
¿Qué puerto utiliza el protocolo SNMP para recibir mensajes de TRAP?
✓ RESPUESTA: B
SNMP utiliza UDP 161 para consultas generales y UDP 162 para mensajes de tipo TRAP.
Pregunta 3
¿Cuál es el prefijo IPv6 de tipo Link-Local?
✓ RESPUESTA: C
FE80::/10 es el rango reservado para direcciones de enlace local en IPv6.
Pregunta 4
¿Qué capa del modelo OSI es responsable de determinar la mejor ruta para enviar paquetes?
✓ RESPUESTA: B
La Capa 3 se encarga del enrutamiento lógico y la determinación de rutas.
Pregunta 5
¿Qué estándar de Ethernet Channel utiliza el protocolo LACP?
✓ RESPUESTA: B
LACP es el protocolo estándar de la industria bajo IEEE 802.3ad.
IPv6
Pregunta 6
¿Cuál es el prefijo reservado para las direcciones IPv6 de tipo Link-Local?
✓ Respuesta: C
Las direcciones Link-Local (FE80::/10) son obligatorias en todas las interfaces IPv6 y solo funcionan dentro del mismo segmento físico.
WIRELESS
Pregunta 7
¿Cuál es el método que utiliza Wi-Fi para evitar colisiones en el medio inalámbrico?
✓ Respuesta: B
Wi-Fi utiliza CSMA/CA (Carrier Sense Multiple Access with Collision Avoidance), mientras que Ethernet por cable usa CSMA/CD.
AUTOMATIZACIÓN
Pregunta 8
En JSON, ¿qué símbolos se utilizan para encerrar un objeto?
✓ Respuesta: B
En JSON, las llaves { } definen objetos (clave:valor) y los corchetes [ ] definen arreglos o listas.
ROUTING
Pregunta 9
¿Cuál es la Distancia Administrativa (AD) de una ruta estática configurada con una interfaz de salida?
✓ Respuesta: B
Aunque se considere "conectada", una ruta estática mantiene su AD de 1 por defecto en la tabla de enrutamiento de Cisco.
SWITCHING
Pregunta 10
¿Qué sucede si hay una discrepancia (mismatch) en la VLAN nativa entre dos switches conectados por un enlace troncal?
✓ Respuesta: B
Un Native VLAN Mismatch genera logs constantes y es un riesgo de seguridad, ya que el tráfico podría saltar de una VLAN a otra.
SEGURIDAD
Pregunta 11
¿Qué tipo de ACL debe colocarse lo más cerca posible del destino?
✓ Respuesta: A
Las ACL Estándar se colocan cerca del destino porque solo filtran por dirección de origen; si se ponen cerca del origen, podrían bloquear tráfico legítimo hacia otros destinos.
SERVICIOS IP
Pregunta 12
¿Qué función de NAT permite que múltiples hosts internos utilicen una sola dirección IP pública mediante el uso de puertos?
✓ Respuesta: C
PAT (o NAT Overload) utiliza números de puerto únicos para distinguir las conexiones de diferentes hosts internos bajo una misma IP pública.
FUNDAMENTOS
Pregunta 13
¿Qué capa del modelo OSI se encarga del direccionamiento físico (MAC addresses)?
✓ Respuesta: B
La Capa 2 gestiona el acceso al medio y utiliza las direcciones MAC para la entrega de tramas.
WIRELESS
Pregunta 14
¿Cuál es el protocolo de seguridad inalámbrica más robusto disponible actualmente?
✓ Respuesta: D
WPA3 es el estándar más reciente y seguro, introduciendo mejoras como el protocolo SAE contra ataques de fuerza bruta.
AUTOMATIZACIÓN
Pregunta 15
En una arquitectura SDN, ¿cómo se llama la interfaz que conecta el controlador con las aplicaciones (plano de aplicación)?
✓ Respuesta: B
La interfaz Northbound comunica al controlador con las aplicaciones, usualmente mediante APIs REST. La Southbound comunica al controlador con los dispositivos físicos.
ROUTING
Pregunta 16
En OSPF, ¿cuál es el estado en el que los routers eligen al DR (Designated Router)?
✓ Respuesta: B
En el estado 2-Way, los routers han establecido comunicación bidireccional y proceden a elegir al DR y BDR si el medio es multiacceso.

✓ RESPUESTA: C
JSON es el formato preferido para las APIs modernas debido a su ligereza.
Pregunta 50
¿Cuál es el rol de un controlador SDN en una arquitectura de red?
✓ RESPUESTA: B
En SDN, el controlador centraliza la lógica del plano de control.
