Pregunta 1
¿Qué valor de Distancia Administrativa (AD) se asigna por defecto a una ruta estática en un router Cisco?
✓ RESPUESTA: B
Las rutas estáticas tienen AD 1. Las interfaces conectadas directamente tienen AD 0.
Pregunta 2
¿Qué puerto TCP utiliza por defecto el protocolo SSH para comunicaciones seguras?
✓ RESPUESTA: B
SSH usa el puerto TCP 22. Telnet usa el 23 y FTP el 21.
Pregunta 3
¿Cuál es la máscara de subred abreviada (/prefijo) para 255.255.255.224?
✓ RESPUESTA: B
/27 corresponde a 224 (128+64+32).
Pregunta 100
En la arquitectura SDN, ¿qué protocolo se utiliza comúnmente para la comunicación del plano de control hacia el plano de datos (Southbound Interface)?
✓ RESPUESTA: B
OpenFlow es el estándar más conocido para interfaces Southbound en redes definidas por software.
