Spanning Tree Protocol

STP 802.1D · RSTP 802.1w · PVST+ · Rapid PVST+ · Prevención de Bucles L2

⚠️ ¿Por Qué Necesitamos STP?

En redes con redundancia física (múltiples cables entre switches), sin ningún protocolo de control se forman bucles de Capa 2. Un bucle L2 destruye la red en segundos. STP (IEEE 802.1D) resuelve esto bloqueando selectivamente algunos puertos, creando una topología libre de bucles.

📋 BPDU y Bridge ID

Los BPDUs (Bridge Protocol Data Units) son mensajes que intercambian los switches para elegir el Root Bridge. Se envían cada 2 segundos (Hello Time) hacia la dirección multicast 01:80:C2:00:00:00.
El Bridge ID (BID) tiene 8 bytes: Bridge Priority (2 bytes, default 32768) + MAC Address (6 bytes). El switch con el BID más bajo gana la elección del Root Bridge.
Timer STPValor DefaultDescripción
Hello Time2 segundosIntervalo entre BPDUs del Root Bridge
Forward Delay15 segundosTiempo en cada estado Listening y Learning
Max Age20 segundosTiempo sin recibir BPDU antes de reconfigurar

🗳️ Los 4 Pasos de la Elección STP

  1. 01
    Elegir el ROOT BRIDGE
    Switch con el BID más bajo. 1°) menor Priority → 2°) menor MAC. Todos sus puertos son Designated (Forwarding).
  2. 02
    Elegir el ROOT PORT en cada switch no-Root
    UN solo Root Port por switch: el de menor costo acumulado hacia el Root Bridge. Siempre en Forwarding.
  3. 03
    Elegir DESIGNATED PORTS en cada segmento
    UN Designated Port por segmento (enlace): el de menor costo hacia el Root Bridge. En Forwarding.
  4. 04
    Puertos restantes → ALTERNATE/BLOCKED
    Descarta frames de datos. Solo escucha BPDUs. Listo para activarse si falla el enlace principal.
🎯
Port Costs: 10 Mbps=100 · FastEthernet=19 · GigabitEthernet=4 · 10 Gbps=2. Mayor velocidad = menor costo = camino preferido.

⏱️
El gran problema: STP 802.1D tarda 30–50 segundos para converger (Listening 15s + Learning 15s). RSTP lo reduce a 1–2 segundos.

🎯
CCNA Examen: El default en switches Cisco modernos (2960, 3750) es Rapid PVST+. RSTP reduce los 5 estados a 3: Discarding, Learning, Forwarding. Agrega nuevos roles: Alternate Port (backup del RP) y Backup Port.

⌨️ Configuración STP en Cisco IOS

ROOT BRIDGE Switch(config)# spanning-tree mode rapid-pvst ! Default moderno Cisco Switch(config)# spanning-tree vlan 10 priority 0 ! Root Bridge absoluto Switch(config)# spanning-tree vlan 10 root primary ! Macro - Root primario Switch(config)# spanning-tree vlan 10 root secondary ! Macro - Root backup
PORTFAST + SEGURIDAD Switch(config-if)# spanning-tree portfast ! Solo en puertos de PCs Switch(config-if)# spanning-tree bpduguard enable ! Deshabilita si recibe BPDU Switch(config-if)# spanning-tree guard root ! Evita robo de Root Bridge Switch(config)# spanning-tree portfast default ! PortFast global en access
